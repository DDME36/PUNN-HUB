# Square Root 3 Rule - กฎ √3

## คำนิยาม
**กฎ √3 (Square Root 3 Rule)** คือกฎสำคัญในระบบ [[Three-Phase-System]] ที่ใช้ √3 (≈ 1.732) ในการแปลงค่าระหว่างแรงดันสาย-เฟส และกระแสสาย-เฟส

## ค่า √3
$$\sqrt{3} = 1.732050808...$$
**ใช้ 1.732 ในการคำนวณ**

## สูตรหลัก

### Star Connection (Y)
$$V_{line} = \sqrt{3} \times V_{phase}$$
$$I_{line} = I_{phase}$$

### Delta Connection (Δ)
$$V_{line} = V_{phase}$$
$$I_{line} = \sqrt{3} \times I_{phase}$$

### กำลังไฟฟ้า 3 เฟส
$$P_{3\phi} = \sqrt{3} \times V_L \times I_L \times \cos\theta$$
$$S_{3\phi} = \sqrt{3} \times V_L \times I_L$$
$$Q_{3\phi} = \sqrt{3} \times V_L \times I_L \times \sin\theta$$

## ที่มาของ √3

### จากเรขาคณิต
```
เฟส 3 ตัวแยกกัน 120°
เมื่อรวมเป็นเวกเตอร์

V_L = 2 × V_P × cos(30°)
V_L = 2 × V_P × (√3/2)
V_L = √3 × V_P
```

### จากสามเหลี่ยมด้านเท่า
- มุมแต่ละมุม = 60°
- ความสูง = (√3/2) × ด้าน
- ความสัมพันธ์ระหว่างเฟส = √3

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ ระบบ 3 เฟส
- [[Three-Phase-System]] → ใช้ [[Square-Root-3-Rule]] เสมอ
- [[Star-Connection]] → $V_L = \sqrt{3} \times V_P$
- [[Delta-Connection]] → $I_L = \sqrt{3} \times I_P$

## ตัวอย่างการใช้งาน

### ตัวอย่าง 1: Star - หาแรงดันสาย
```
V_phase = 220V (เฟสกับนิวทรัล)
V_line = √3 × V_phase
V_line = 1.732 × 220 = 381V ≈ 380V
```

### ตัวอย่าง 2: Delta - หากระแสสาย
```
I_phase = 50A (กระแสในขดลวด)
I_line = √3 × I_phase
I_line = 1.732 × 50 = 86.6A
```

### ตัวอย่าง 3: หากำลัง 3 เฟส
```
V_L = 380V, I_L = 100A, PF = 0.8
P = √3 × V_L × I_L × cosθ
P = 1.732 × 380 × 100 × 0.8
P = 52,652W ≈ 52.7 kW
```

### ตัวอย่าง 4: เปรียบเทียบ 1 เฟส vs 3 เฟส
```
1 เฟส: P = V × I × cosθ = 220 × 100 × 0.8 = 17.6 kW
3 เฟส: P = √3 × V × I × cosθ = 1.732 × 220 × 100 × 0.8 = 30.5 kW

3 เฟส ≠ 1 เฟส × 3
3 เฟส = 1 เฟส × √3 (ต่อเฟสเดียว)
```

## ข้อผิดพลาดที่พบบ่อย (CRITICAL!)

### ❌ ข้อผิดพลาดที่ 1: ลืมคูณ √3
```
ผิด: P = V_L × I_L × cosθ
ถูก: P = √3 × V_L × I_L × cosθ
```

### ❌ ข้อผิดพลาดที่ 2: คิดว่า 380V = 220V × 2
```
ผิด: 380 = 220 × 2 = 440V (ผิด!)
ถูก: 380 = 220 × √3 = 220 × 1.732 = 381V ≈ 380V
```

### ❌ ข้อผิดพลาดที่ 3: ใช้ √3 ผิดที่
```
Star: V_L = √3 × V_P ✓ (ถูก)
      I_L = √3 × I_P ✗ (ผิด! I_L = I_P)

Delta: V_L = √3 × V_P ✗ (ผิด! V_L = V_P)
       I_L = √3 × I_P ✓ (ถูก)
```

### ❌ ข้อผิดพลาดที่ 4: คิดว่า 3 เฟส = 1 เฟส × 3
```
ผิด: P_3φ = P_1φ × 3
ถูก: P_3φ = P_1φ × √3 (ถ้าเทียบต่อเฟสเดียว)
หรือ: P_3φ = 3 × V_P × I_P × cosθ (ถ้าใช้ค่าเฟส)
```

## เทคนิคจำ

### 🎯 จำง่ายๆ
1. **เห็น "3 เฟส" → คิด √3**
2. **Star → √3 อยู่ที่ V**
3. **Delta → √3 อยู่ที่ I**
4. **กำลัง 3 เฟส → มี √3 เสมอ**

### 🎯 ตารางสรุป
| การต่อ | V_L vs V_P | I_L vs I_P |
|---|---|---|
| Star (Y) | V_L = √3 × V_P | I_L = I_P |
| Delta (Δ) | V_L = V_P | I_L = √3 × I_P |

## การตรวจสอบ
```
ถ้าคำนวณแล้วได้:
- 380V จาก 220V → ถูก (220 × 1.732 = 381)
- 440V จาก 220V → ผิด! (คูณ 2 แทน √3)
- 660V จาก 380V → ถูก (380 × 1.732 = 658)
```
