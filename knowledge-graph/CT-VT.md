# CT-VT - หม้อแปลงวัด

## คำนิยาม
**CT (Current Transformer)** และ **VT/PT (Voltage Transformer/Potential Transformer)** คือหม้อแปลงพิเศษที่ใช้วัดกระแสและแรงดันสูง โดยแปลงเป็นค่าต่ำที่ปลอดภัย

## CT - Current Transformer

### หลักการ
- แปลงกระแสสูง → กระแสต่ำ (มาตรฐาน 5A หรือ 1A)
- ต่อ **อนุกรม** กับวงจร
- ฝั่งทุติยภูมิ **ห้ามเปิดวงจร!**

### อัตราส่วน (Ratio)
```
CT 100/5A
- ปฐมภูมิ: 100A
- ทุติยภูมิ: 5A
- Ratio = 100/5 = 20:1
```

### สัญลักษณ์
```
Primary  ⊕  Secondary
  ───────║║─────────
         ║║
    I₁   ║║   I₂
```

### การคำนวณ
$\frac{I_1}{I_2} = \frac{N_2}{N_1}$

ตัวอย่าง:
```
CT 200/5A
I₁ = 150A (จริง)
I₂ = 150 × (5/200) = 3.75A (วัดได้)
```

### Class Accuracy
- **Class 0.5**: ความคลาดเคลื่อน ±0.5% (วัดพลังงาน)
- **Class 1**: ±1% (วัดทั่วไป)
- **Class 5P**: ±5% (ป้องกัน)

### ⚠️ อันตราย!
```
ห้ามเปิดวงจรฝั่งทุติยภูมิ!

เหตุผล:
- ไม่มีกระแส I₂ → ไม่มีสนามแม่เหล็กต้าน
- สนามแม่เหล็ก I₁ เต็มที่
- เหนี่ยวนำ V₂ สูงมาก (หลักพัน Volt!)
- ไฟดูด, แกนเหล็กร้อน, อันตราย!

แก้ไข: Short ฝั่งทุติยภูมิก่อนถอดมิเตอร์
```

## VT/PT - Voltage Transformer

### หลักการ
- แปลงแรงดันสูง → แรงดันต่ำ (มาตรฐาน 110V)
- ต่อ **ขนาน** กับวงจร
- ฝั่งทุติยภูมิ **ห้าม Short!**

### อัตราส่วน (Ratio)
```
VT 22000/110V
- ปฐมภูมิ: 22 kV
- ทุติยภูมิ: 110V
- Ratio = 22000/110 = 200:1
```

### สัญลักษณ์
```
Primary  ||  Secondary
  )))    ||    (((
  V₁     ||    V₂
```

### การคำนวณ
$\frac{V_1}{V_2} = \frac{N_1}{N_2}$

ตัวอย่าง:
```
VT 22000/110V
V₁ = 20 kV (จริง)
V₂ = 20,000 × (110/22,000) = 100V (วัดได้)
```

### Class Accuracy
- **Class 0.5**: ±0.5% (วัดพลังงาน)
- **Class 1**: ±1% (วัดทั่วไป)
- **Class 3P**: ±3% (ป้องกัน)

### ⚠️ อันตราย!
```
ห้าม Short ฝั่งทุติยภูมิ!

เหตุผล:
- V₂ = 0 → I₂ สูงมาก
- I₁ สูงตาม
- VT ไหม้!

แก้ไข: ใช้ฟิวส์ป้องกัน
```

## การใช้งานร่วมกัน

### วัดกำลังไฟฟ้า 3 เฟส
```
CT: 100/5A
VT: 22000/110V

P_จริง = √3 × V_วัด × I_วัด × PF × CT_ratio × VT_ratio
P_จริง = √3 × 110 × 5 × 0.9 × 20 × 200
P_จริง = 3,435 kW
```

### มิเตอร์วัดหน่วย ([[Energy-Meter]])
```
CT: 200/5A (Ratio = 40)
VT: 22000/110V (Ratio = 200)

หน่วยจริง = หน่วยที่วัด × 40 × 200
หน่วยจริง = หน่วยที่วัด × 8,000
```

## ความแตกต่าง CT vs VT

| หัวข้อ | CT | VT |
|---|---|---|
| วัด | กระแส | แรงดัน |
| ต่อ | อนุกรม | ขนาน |
| ห้าม | เปิดวงจร | Short วงจร |
| มาตรฐาน | 5A, 1A | 110V |
| อันตราย | แรงดันสูง | กระแสสูง |

## การเลือกใช้

### เลือก CT
```
กระแสโหลดสูงสุด = 150A
เลือก CT ≥ 150A
→ CT 200/5A (120% ของโหลด)
```

### เลือก VT
```
แรงดันระบบ = 22 kV
เลือก VT ตามแรงดันระบบ
→ VT 22000/110V
```

### Burden (ภาระ)
- **Burden**: โหลดที่ต่อฝั่งทุติยภูมิ (VA)
- ต้องไม่เกิน Burden Rating
- เช่น: 15 VA, 30 VA

## การต่อลงดิน
- **CT**: ต่อลงดินฝั่งทุติยภูมิ (ป้องกันไฟดูด)
- **VT**: ต่อลงดินฝั่งทุติยภูมิ (ป้องกันไฟรั่ว)

## ข้อผิดพลาดที่พบบ่อย

### ❌ เปิดวงจร CT
```
ผิด: ถอดมิเตอร์ทันที
ถูก: Short ฝั่งทุติยภูมิก่อน
     ใช้ Terminal Block แบบ Short
```

### ❌ Short วงจร VT
```
ผิด: ต่อสาย V₂ ลัดวงจร
ถูก: ต่อผ่านมิเตอร์หรือโหลด
     ใช้ฟิวส์ป้องกัน
```

### ❌ สลับ CT กับ VT
```
ผิด: ใช้ CT วัดแรงดัน
ถูก: CT → กระแส (อนุกรม)
     VT → แรงดัน (ขนาน)
```

### ❌ คำนวณผิด
```
ผิด: P = V_วัด × I_วัด
ถูก: P = V_วัด × I_วัด × CT_ratio × VT_ratio

ตัวอย่าง:
V_วัด = 110V, I_วัด = 4A
CT = 100/5, VT = 22000/110
P = 110 × 4 × 20 × 200 = 1,760 kW
```

2. **Polarity Test**: ตรวจขั้ว
3. **Burden Test**: ตรวจภาระ
4. **Insulation Test**: ตรวจฉนวน
