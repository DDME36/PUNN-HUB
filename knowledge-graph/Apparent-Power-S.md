# Apparent Power (S) - กำลังปรากฏ

## คำนิยาม
**กำลังปรากฏ (Apparent Power)** คือกำลังไฟฟ้ารวมทั้งหมดที่ส่งไปยังโหลด ประกอบด้วยทั้ง [[Real-Power-P]] และ [[Reactive-Power-Q]]

## สัญลักษณ์
- **S** = Apparent Power
- หน่วย: **VA (Volt-Ampere)**, **kVA**, **MVA**

## สูตรพื้นฐาน

### 1 เฟส
$$S = V \times I$$

### 3 เฟส
$$S = \sqrt{3} \times V_L \times I_L$$

### จากสามเหลี่ยมกำลัง
$$S = \sqrt{P^2 + Q^2}$$

$$S = \frac{P}{PF} = \frac{P}{\cos\theta}$$

## ความสัมพันธ์กับแนวคิดอื่น

### 🔺 สามเหลี่ยมกำลัง
```
      S
     /|
    / |
   /  | Q
  /   |
 /θ___|
    P
```

- [[Apparent-Power-S]] = ด้านตรงข้ามมุมฉาก
- [[Real-Power-P]] = ด้านประชิด
- [[Reactive-Power-Q]] = ด้านตรงข้าม
- [[Power-Factor-PF]] = cos θ = P/S

### ⚡ รูปแบบเชิงซ้อน
$$\vec{S} = P + jQ$$

$$|\vec{S}| = \sqrt{P^2 + Q^2}$$

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หา S จาก V และ I
```
V = 220V, I = 10A
S = V × I = 220 × 10 = 2,200 VA = 2.2 kVA
```

### ตัวอย่าง 2: หา S จาก P และ PF
```
P = 10 kW, PF = 0.8
S = P / PF = 10 / 0.8 = 12.5 kVA
```

### ตัวอย่าง 3: หา S จาก P และ Q
```
P = 8 kW, Q = 6 kVAR
S = √(P² + Q²) = √(8² + 6²)
S = √(64 + 36) = √100 = 10 kVA
```

### ตัวอย่าง 4: ระบบ 3 เฟส
```
V_L = 380V, I_L = 100A
S = √3 × V_L × I_L
S = 1.732 × 380 × 100
S = 65,816 VA ≈ 65.8 kVA
```

## การใช้งาน

### เนมเพลต
- [[Transformer]]: ระบุเป็น kVA (เช่น 100 kVA)
- [[Motor]]: ระบุเป็น kW (Output) แต่คำนวณ S ได้จาก V, I

### การเลือกอุปกรณ์
```
โหลด 10 kW, PF = 0.7
S = 10 / 0.7 = 14.3 kVA

ต้องเลือก:
- หม้อแปลง ≥ 14.3 kVA
- สาย, เบรกเกอร์ รองรับ I = S/(√3 × V)
```

## ความแตกต่าง kW vs kVA

| หน่วย | ความหมาย | ใช้กับ |
|---|---|---|
| kW | กำลังจริงที่ใช้งานได้ | โหลด Resistive, มอเตอร์ (Output) |
| kVA | กำลังปรากฏทั้งหมด | หม้อแปลง, เครื่องกำเนิด |
| kVAR | กำลังรีแอคทีฟ | Capacitor Bank |

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** kW = kVA  
✅ **ถูก:** kW = kVA × PF

❌ **ผิด:** S = P + Q (บวกตรงๆ)  
✅ **ถูก:** S = √(P² + Q²) (ทฤษฎีพีทาโกรัส)

❌ **ผิด:** หม้อแปลง 100 kVA ให้กำลัง 100 kW  
✅ **ถูก:** ให้ได้ P = S × PF (ขึ้นกับ PF ของโหลด)
