# Synchronous Speed (Ns) - ความเร็วซิงโครนัส

## คำนิยาม
**ความเร็วซิงโครนัส (Synchronous Speed)** คือความเร็วของสนามแม่เหล็กหมุนใน [[Motor]] ซึ่งขึ้นกับความถี่และจำนวนขั้ว

## สัญลักษณ์
- **Ns** = Synchronous Speed
- หน่วย: **rpm** (รอบต่อนาที)

## สูตร
$$N_s = \frac{120f}{P}$$

- f = [[Frequency-f]] (Hz)
- P = จำนวนขั้ว (poles) - ต้องเป็นเลขคู่

## ความสัมพันธ์กับแนวคิดอื่น

### ⚙️ มอเตอร์
- [[Synchronous-Speed]] → ความเร็วสนามแม่เหล็ก
- [[Motor]] → ความเร็วโรเตอร์ $N_r < N_s$
- [[Slip]] → $s = \frac{N_s - N_r}{N_s}$

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: มอเตอร์ 4 poles
```
f = 50 Hz, P = 4
N_s = 120f/P = 120 × 50 / 4
N_s = 1,500 rpm
```

### ตัวอย่าง 2: มอเตอร์ 2 poles
```
f = 50 Hz, P = 2
N_s = 120 × 50 / 2 = 3,000 rpm
```

### ตัวอย่าง 3: หาจำนวนขั้ว
```
N_s = 1,500 rpm, f = 50 Hz
P = 120f / N_s = 120 × 50 / 1,500
P = 4 poles
```

## ตารางความเร็วมาตรฐาน (50 Hz)

| Poles | Ns (rpm) | Nr โดยประมาณ |
|---|---|---|
| 2 | 3,000 | 2,850-2,950 |
| 4 | 1,500 | 1,400-1,450 |
| 6 | 1,000 | 950-980 |
| 8 | 750 | 720-740 |

## ข้อสังเกต
- Ns = ความเร็วสนามแม่เหล็ก (คงที่)
- Nr = ความเร็วโรเตอร์ (ช้ากว่า Ns เล็กน้อย)
- Nr < Ns เสมอ (มี [[Slip]])
