# Kirchhoff's Laws - กฎของเคอร์ชอฟฟ์

## คำนิยาม
**กฎของเคอร์ชอฟฟ์ (Kirchhoff's Laws)** คือกฎพื้นฐาน 2 ข้อในการวิเคราะห์วงจรไฟฟ้า ใช้หาค่า [[Current-I]] และ [[Voltage-V]] ในวงจร

## กฎทั้ง 2 ข้อ

### 1. KCL - Kirchhoff's Current Law (กฎกระแส)
**"ผลรวมกระแสที่เข้าและออกจากจุดต่อ (Node) = 0"**

$\sum I_{in} = \sum I_{out}$

หรือ

$\sum I = 0$

### 2. KVL - Kirchhoff's Voltage Law (กฎแรงดัน)
**"ผลรวมแรงดันตกในวงปิด (Loop) = 0"**

$\sum V = 0$

## หลักการ

### KCL - อนุรักษ์ประจุ
- ประจุไม่สูญหาย ไม่เกิดขึ้นเอง
- กระแสเข้า = กระแสออก
- ใช้ที่ **จุดต่อ (Node)**

### KVL - อนุรักษ์พลังงาน
- พลังงานไม่สูญหาย
- แรงดันเพิ่ม = แรงดันตก
- ใช้ใน **วงปิด (Loop)**

## ตัวอย่างการใช้งาน

### ตัวอย่าง 1: KCL ที่จุดต่อ
```
จุดต่อ A:
I₁ = 5A (เข้า)
I₂ = 3A (เข้า)
I₃ = ? (ออก)

KCL: I₁ + I₂ = I₃
I₃ = 5 + 3 = 8A
```

### ตัวอย่าง 2: KVL ในวงปิด
```
วงจรอนุกรม:
V_source = 12V
V_R1 = 4V
V_R2 = ?

KVL: V_source - V_R1 - V_R2 = 0
12 - 4 - V_R2 = 0
V_R2 = 8V
```

### ตัวอย่าง 3: วงจรซับซ้อน
```
วงจร 2 Loop:
Loop 1: V₁ - I₁R₁ - I₃R₃ = 0
Loop 2: -I₂R₂ - I₃R₃ + V₂ = 0

Node A: I₁ + I₂ = I₃

แก้สมการ 3 ตัวแปร
```

## การประยุกต์ใช้

### 1. วงจรอนุกรม ([[Series-Connection]])
```
KVL: V_total = V₁ + V₂ + V₃
KCL: I_total = I₁ = I₂ = I₃
```

### 2. วงจรขนาน ([[Parallel-Connection]])
```
KVL: V_total = V₁ = V₂ = V₃
KCL: I_total = I₁ + I₂ + I₃
```

### 3. วงจรสามเฟส ([[Three-Phase-System]])
```
KCL ที่ Neutral:
I_N = I_A + I_B + I_C

ถ้าสมดุล: I_N = 0
```

## เทคนิคการใช้

### เครื่องหมาย (Sign Convention)
```
KCL:
+ กระแสเข้าจุดต่อ
- กระแสออกจากจุดต่อ

KVL (เดินตามเข็มนาฬิกา):
+ แรงดันเพิ่ม (แบตเตอรี่ +→-)
- แรงดันตก (ตัวต้านทาน)
```

### ขั้นตอนการแก้ปัญหา
1. กำหนดทิศทางกระแส (สุ่มได้)
2. เขียนสมการ KCL ที่จุดต่อ
3. เขียนสมการ KVL ในวงปิด
4. แก้สมการหาค่าที่ต้องการ
5. ถ้าได้ค่าลบ = ทิศทางตรงข้าม

## ตัวอย่างโจทย์ยาก

### โจทย์: วงจร 2 แหล่งจ่าย
```
V₁ = 12V, V₂ = 6V
R₁ = 2Ω, R₂ = 3Ω, R₃ = 4Ω

หา: I₁, I₂, I₃

วิธีทำ:
1. กำหนด I₁, I₂, I₃
2. Node A: I₁ + I₂ = I₃
3. Loop 1: 12 - 2I₁ - 4I₃ = 0
4. Loop 2: 6 - 3I₂ - 4I₃ = 0
5. แก้สมการ 3 ตัว

ผลลัพธ์:
I₁ = 2.4A
I₂ = 0.8A
I₃ = 3.2A
```

## ข้อผิดพลาดที่พบบ่อย

### ❌ เครื่องหมายผิด
```
ผิด: V₁ + V₂ + V₃ = 0 (ทุกตัวบวก)
ถูก: V_source - V_R1 - V_R2 = 0
     (แหล่งจ่าย +, ตัวต้านทาน -)
```

### ❌ นับซ้ำ
```
ผิด: ใช้ KCL ทุกจุดต่อ (สมการซ้ำ!)
ถูก: ใช้ n-1 สมการ (n = จำนวนจุดต่อ)
```

### ❌ ลืมทิศทาง
```
ผิด: ไม่กำหนดทิศทางกระแส
ถูก: กำหนดทิศทางก่อน (ผิดก็ได้ค่าลบ)
```

### ❌ สับสน KCL กับ KVL
```
KCL: ใช้ที่จุดต่อ (Node) - กระแส
KVL: ใช้ในวงปิด (Loop) - แรงดัน
```

## การใช้ใน AC Circuit

### วงจร AC
```
KCL: ∑Ĩ = 0 (Complex Number)
KVL: ∑Ṽ = 0 (Complex Number)

ใช้ [[Complex-Numbers-AC]] คำนวณ
```

### ตัวอย่าง AC
```
Ṽ = 220∠0° V
Z₁ = 3 + j4 Ω
Z₂ = 5 - j2 Ω

KVL: Ṽ - Ĩ(Z₁ + Z₂) = 0
Ĩ = Ṽ / (Z₁ + Z₂)
Ĩ = 220∠0° / (8 + j2)
Ĩ = 26.8∠-14° A
```
