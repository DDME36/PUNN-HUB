# Step Voltage & Touch Voltage - แรงดันก้าวและแรงดันสัมผัส

## คำนิยาม

### Step Voltage (แรงดันก้าว)
**แรงดันก้าว** คือความต่างศักย์ระหว่างเท้าทั้งสองข้าง (ห่างกัน 1 เมตร) เมื่อมีกระแสลงดิน

### Touch Voltage (แรงดันสัมผัส)
**แรงดันสัมผัส** คือความต่างศักย์ระหว่างมือที่จับอุปกรณ์กับเท้าที่ยืนพื้น

## ภาพประกอบ

### Step Voltage
```
      กระแสลงดิน ↓
           ⚡
    ═══════╪═══════  พื้นดิน
           │
    ●─────●─────●   ระดับศักย์
    │  1m │  1m │
    
    👤 คน (เท้า 2 ข้าง)
    ├─────┤ 1 m
    V_step = ความต่างศักย์
```

### Touch Voltage
```
      อุปกรณ์ไฟฟ้า
         ┌─┐
         │⚡│ ← ไฟรั่ว
         └─┘
          │
      👤 คน
      ✋ มือจับ
      │
      │ V_touch
      │
    ═══╪═══  พื้นดิน
       🦶 เท้า
```

## สูตรการคำนวณ

### Step Voltage
$V_{step} = \rho \times I_g \times K_s$

- ρ = ความต้านทานจำเพาะของดิน (Ω·m)
- I_g = กระแสลงดิน (A)
- K_s = ค่าคงที่ (ขึ้นกับระยะห่าง)

### Touch Voltage
$V_{touch} = \rho \times I_g \times K_t$

- K_t = ค่าคงที่ (ขึ้นกับระบบลงดิน)

## ค่าที่อันตราย

### แรงดันที่อันตราย (50 Hz)
| สภาพ | V_step (V) | V_touch (V) |
|---|---|---|
| แห้ง | 116 | 50 |
| เปียก | 58 | 25 |

### กระแสที่อันตราย
| ระดับ | กระแส (mA) | ผลกระทบ |
|---|---|---|
| รับรู้ได้ | 1 | รู้สึกเสียว |
| ปล่อยไม่ได้ | 10-15 | กล้ามเนื้อหดเกร็ง |
| หายใจไม่ได้ | 30 | หายใจลำบาก |
| หัวใจหยุด | 100 | เสียชีวิต |

## ปัจจัยที่มีผล

### 1. ความต้านทานดิน (ρ)
| ประเภทดิน | ρ (Ω·m) |
|---|---|
| ดินเหนียวเปียก | 10-50 |
| ดินทั่วไป | 50-200 |
| ดินทราย | 200-1000 |
| หินแกรนิต | 1000-10000 |

### 2. ความต้านทานร่างกาย
```
มือแห้ง: 1,000-100,000 Ω
มือเปียก: 500-1,000 Ω
มือเปียกมาก: 200-500 Ω
```

### 3. เวลาสัมผัส
```
< 0.1 วินาที: ปลอดภัย
0.1-1 วินาที: อันตราย
> 1 วินาที: เสียชีวิต
```

## การป้องกัน

### 1. Ground Grid (ตาข่ายลงดิน)
```
┌─────┬─────┬─────┐
│     │     │     │
├─────┼─────┼─────┤
│     │     │     │
├─────┼─────┼─────┤
│     │     │     │
└─────┴─────┴─────┘

ลดความต่างศักย์บนพื้น
```

### 2. Equipotential Bonding
```
อุปกรณ์ทั้งหมดต่อลงดินเดียวกัน
→ ศักย์เท่ากัน
→ ไม่มีความต่างศักย์
```

### 3. RCD/RCCB
```
ตัดไฟภายใน 0.03 วินาที
→ ป้องกันไฟดูด
```

### 4. Insulation Mat
```
ปูเสื่อฉนวนบนพื้น
→ เพิ่มความต้านทาน
→ ลดกระแสผ่านร่างกาย
```

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: Step Voltage
```
กระแสลงดิน: 1,000A
ความต้านทานดิน: 100 Ω·m
ระยะห่างเท้า: 1 m

V_step ≈ 100 × 1,000 × 0.1
V_step ≈ 10,000V (อันตรายมาก!)

แก้ไข: ติดตั้ง Ground Grid
→ ลด V_step เหลือ < 116V
```

### ตัวอย่าง 2: Touch Voltage
```
กระแสลงดิน: 500A
ความต้านทานดิน: 50 Ω·m
ความต้านทานลงดิน: 5 Ω

V_touch = 500 × 5 = 2,500V
(อันตรายมาก!)

แก้ไข: ติดตั้ง RCD 30mA
→ ตัดไฟก่อนอันตราย
```

## มาตรฐาน

### IEEE Std 80
- มาตรฐานการออกแบบระบบลงดิน
- คำนวณ Step & Touch Voltage
- กำหนดค่าที่ปลอดภัย

### IEC 61936-1
- มาตรฐานยุโรป
- ระบบลงดินสถานีไฟฟ้า

## ข้อผิดพลาดที่พบบ่อย

### ❌ คิดว่าลงดินแล้วปลอดภัย
```
ผิด: ลงดินแล้ว = ปลอดภัย 100%
ถูก: ต้องคำนวณ Step & Touch Voltage
     ต้องไม่เกินค่าที่กำหนด
```

### ❌ ไม่ใช้ RCD
```
ผิด: มีลงดินดีแล้ว ไม่ต้อง RCD
ถูก: ต้องมี RCD เพื่อตัดไฟเร็ว
     (< 0.03 วินาที)
```

### ❌ ใช้ความต้านทานลงดินสูง
```
ผิด: ลงดิน 50 Ω (ยอมรับได้)
ถูก: ควร < 5 Ω สำหรับความปลอดภัย
```
