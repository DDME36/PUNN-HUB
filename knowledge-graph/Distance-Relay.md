# Distance Relay - รีเลย์ระยะทาง

## คำนิยาม
**รีเลย์ระยะทาง (Distance Relay)** คือรีเลย์ที่ตรวจจับฟอลต์โดยวัด [[Impedance-Z]] ระหว่างตำแหน่งรีเลย์ถึงจุดฟอลต์ ใช้ป้องกันสายส่งไฟฟ้า

## หลักการทำงาน

### การวัด Impedance
$$Z = \frac{V}{I}$$

- V = แรงดันที่รีเลย์
- I = กระแสที่รีเลย์
- Z = อิมพีแดนซ์ที่คำนวณได้

### การตัดสินใจ
```
ถ้า Z < Z_set → ฟอลต์อยู่ในโซนป้องกัน → ตัด
ถ้า Z > Z_set → ฟอลต์อยู่นอกโซน → ไม่ตัด
```

## ประเภท

### 1. Impedance Relay
```
วงกลมบน R-X Diagram
ศูนย์กลางที่จุดกำเนิด
รัศมี = Z_set
```

#### คุณสมบัติ
- ตรวจจับทุกทิศทาง
- ไม่เลือกทิศทาง
- ไม่นิยมใช้แล้ว

### 2. Mho Relay (Admittance Relay)
```
วงกลมผ่านจุดกำเนิด
ศูนย์กลางบนแกน R
```

#### คุณสมบัติ
- เลือกทิศทางได้
- ทนแรงดันแกว่งได้ดี
- นิยมใช้มากที่สุด

### 3. Reactance Relay
```
เส้นตรงขนานแกน R
X = X_set
```

#### คุณสมบัติ
- ไม่ขึ้นกับ R
- เหมาะกับสายสั้น
- ใช้ร่วมกับ Mho

## โซนป้องกัน (Protection Zones)

### Zone 1 (Instantaneous)
- ป้องกัน 80-85% ของสายส่ง
- ตัดทันที (< 0.1 s)
- ไม่มี Time Delay

### Zone 2 (Time Delay)
- ป้องกัน 120-150% ของสายส่ง
- หน่วงเวลา 0.3-0.5 s
- Backup สำหรับสายถัดไป

### Zone 3 (Backup)
- ป้องกัน 200-250% ของสายส่ง
- หน่วงเวลา 1.0-1.5 s
- Backup ระยะไกล

## R-X Diagram

### แกน
- แกน X: Reactance (X)
- แกน R: Resistance (R)
- จุดกำเนิด: ตำแหน่งรีเลย์

### การแสดงผล
```
สายส่ง: เส้นตรงจากจุดกำเนิด
  ความชัน = X/R ของสาย
  ความยาว = Z ของสาย

โซนป้องกัน: วงกลม/เส้นตรง
  Zone 1 < Zone 2 < Zone 3
```

## การตั้งค่า

### Zone 1
$$Z_1 = 0.8 \times Z_{line}$$

### Zone 2
$$Z_2 = Z_{line} + 0.5 \times Z_{next}$$

### Zone 3
$$Z_3 = Z_{line} + Z_{next} + 0.25 \times Z_{3rd}$$

## ข้อดี
- ไม่ขึ้นกับขนาดกระแสลัดวงจร
- ป้องกันได้ทุกชนิดฟอลต์
- เหมาะกับสายส่งยาว
- มี Backup ในตัว

## ข้อเสีย
- ราคาแพง
- ซับซ้อน
- ต้องการ CT และ VT
- ได้รับผลกระทบจาก Power Swing

## ปัญหาที่พบบ่อย

### 1. Underreach
- ตรวจจับได้น้อยกว่าที่ตั้ง
- สาเหตุ: ความต้านทานฟอลต์สูง

### 2. Overreach
- ตรวจจับได้มากกว่าที่ตั้ง
- สาเหตุ: Infeed Effect

### 3. Power Swing
- แรงดันและกระแสแกว่ง
- รีเลย์อาจตัดผิด
- แก้ไข: ใช้ Power Swing Blocking

## ANSI Code
