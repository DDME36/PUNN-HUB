# Capacitive Reactance (XC) - รีแอคแตนซ์ความจุ

## คำนิยาม
**รีแอคแตนซ์ความจุ (Capacitive Reactance)** คือความต้านทานที่เกิดจาก [[Capacitor-C]] ในวงจร AC ซึ่งลดลงตามความถี่

## สัญลักษณ์
- **XC** = Capacitive Reactance
- หน่วย: **Ohm (Ω)**

## สูตร
$$X_C = \frac{1}{2\pi f C} = \frac{1}{\omega C}$$

- f = [[Frequency-f]] (Hz)
- C = ความจุไฟฟ้า (F)
- ω = ความถี่เชิงมุม (rad/s)

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ พื้นฐาน
- [[Capacitor-C]] → สร้าง [[Capacitive-Reactance-XC]]
- [[Capacitive-Reactance-XC]] → ส่วนหนึ่งของ [[Reactance]]
- [[Capacitive-Reactance-XC]] → ส่วน $-jX_C$ ใน [[Impedance-Z]]

### 🔺 กำลังและเฟส
- [[Capacitive-Reactance-XC]] → สร้าง [[Reactive-Power-Q]] นำหน้า
- กระแสนำหน้าแรงดัน 90°
- [[Power-Factor-PF]] นำหน้า (Leading)

### ⚙️ การใช้งาน
- [[Capacitor-Bank]] → ใช้ XC แก้ PF
- [[Power-Factor-Correction]]
- Filter, Coupling

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หา XC
```
C = 100 μF = 0.0001 F, f = 50 Hz
X_C = 1 / (2πfC)
X_C = 1 / (2 × 3.14159 × 50 × 0.0001)
X_C = 31.8 Ω
```

### ตัวอย่าง 2: ผลของความถี่
```
C = 100 μF
f = 50 Hz → X_C = 31.8 Ω
f = 60 Hz → X_C = 26.5 Ω
f = 100 Hz → X_C = 15.9 Ω
```

### ตัวอย่าง 3: หา C
```
X_C = 100 Ω, f = 50 Hz
C = 1 / (2πfX_C)
C = 1 / (2π × 50 × 100)
C = 0.0000318 F = 31.8 μF
```

## ความสัมพันธ์กับความถี่

| ความถี่ | XC | พฤติกรรม |
|---|---|---|
| DC (0 Hz) | ∞ Ω | Open circuit |
| ต่ำ (50 Hz) | สูง | ต้านมาก |
| สูง (1 MHz) | ต่ำมาก | ต้านน้อย |

**สรุป:** ยิ่งความถี่สูง → XC ยิ่งต่ำ (สัดส่วนกลับกัน)

## เฟสเซอร์
```
I นำหน้า V 90°
หรือ V ล้าหลัง I 90°

V = V∠0°
I = I∠+90°
```

## การใช้แก้ Power Factor
```
มอเตอร์มี X_L สูง → Q ล้าหลัง
ติดตั้ง Capacitor → X_C → Q นำหน้า
Q ล้าหลัง + Q นำหน้า = หักล้างกัน
→ PF ดีขึ้น
```

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** XC คงที่  
✅ **ถูก:** XC ขึ้นกับความถี่ (XC ∝ 1/f)

❌ **ผิด:** ใน DC, XC = 0  
✅ **ถูก:** ใน DC, XC = ∞ (ตัดวงจร)

❌ **ผิด:** XC ทำให้กระแสล้าหลัง  
✅ **ถูก:** XC ทำให้กระแสนำหน้า 90°

❌ **ผิด:** XC = 2πfC  
✅ **ถูก:** XC = 1/(2πfC) (มี 1 หาร!)
