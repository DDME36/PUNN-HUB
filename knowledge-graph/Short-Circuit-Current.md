# Short-Circuit Current - กระแสลัดวงจร

## คำนิยาม
**กระแสลัดวงจร (Short-Circuit Current)** คือกระแสที่ไหลเมื่อเกิดการลัดวงจร (เฟสชนกัน หรือเฟสชนดิน)

## สูตร
$$I_{sc} = \frac{V}{Z}$$

### จาก %Z ของหม้อแปลง
$$I_{sc} = \frac{I_{rated}}{\%Z} \times 100$$

## ตัวอย่าง
```
หม้อแปลง 100 kVA, 380V, %Z = 5%
I_rated = 100,000/(√3 × 380) = 152A

I_sc = (152 / 5) × 100 = 3,040A
```

## อันตราย
- ความร้อนสูงมาก
- แรงแม่เหล็กสูง
- ไฟไหม้
- ระเบิด

## การป้องกัน
- [[Circuit-Breaker]] ที่รองรับ Isc
- Fuse
- [[Overload-Relay]]
- ระยะห่างเพียงพอ
