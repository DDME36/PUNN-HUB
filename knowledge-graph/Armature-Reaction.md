# Armature Reaction - ปฏิกิริยาอาร์มาเจอร์

## คำนิยาม
**ปฏิกิริยาอาร์มาเจอร์ (Armature Reaction)** คือผลกระทบของสนามแม่เหล็กที่เกิดจากกระแสใน [[Armature-Winding]] ซึ่งรบกวนสนามแม่เหล็กหลักจาก [[Field-Winding]]

## ผลกระทบ

### 1. Demagnetization (ลดแม่เหล็ก)
- สนามจากอาร์มาเจอร์ต้านสนามหลัก
- ทำให้ฟลักซ์รวมลดลง
- แรงบิดและ EMF ลดลง

### 2. Cross-Magnetization (แม่เหล็กตัด)
- สนามจากอาร์มาเจอร์ตัดฉากกับสนามหลัก
- บิดเบือนรูปร่างสนามแม่เหล็ก
- ทำให้การกระจายฟลักซ์ไม่สม่ำเสมอ

### 3. Shift of Neutral Axis (เลื่อนแกนนิวทรัล)
- แกนนิวทรัล (จุดที่ฟลักซ์ = 0) เลื่อนตำแหน่ง
- ใน Motor: เลื่อนตามทิศการหมุน
- ใน Generator: เลื่อนทวนทิศการหมุน

## ปัญหาที่เกิด

### Poor Commutation
- แปรงถ่านไม่อยู่บนแกนนิวทรัลที่แท้จริง
- เกิดประกายไฟที่ [[Commutator]]
- แปรงถ่านและ Commutator สึกหรอเร็ว

### ลดประสิทธิภาพ
- ฟลักซ์ลดลง → EMF ลดลง
- แรงบิดลดลง
- ประสิทธิภาพลดลง

## วิธีแก้ไข

### 1. Interpole (ขั้วแม่เหล็กเสริม)
```
ติดตั้งระหว่างขั้วหลัก
ต่ออนุกรมกับอาร์มาเจอร์
สร้างสนามต้าน Armature Reaction
```

#### คุณสมบัติ
- ปรับตามโหลดอัตโนมัติ
- ช่วย Commutation
- ใช้กันมากที่สุด

### 2. Compensating Winding (ขดลวดชดเชย)
```
ฝังในหน้าขั้วหลัก
ต่ออนุกรมกับอาร์มาเจอร์
ต้านสนามจากอาร์มาเจอร์โดยตรง
```

#### คุณสมบัติ
- แก้ปัญหาได้ดีที่สุด
- ราคาแพง
- ใช้กับเครื่องจักรขนาดใหญ่

### 3. Brush Shift (เลื่อนแปรงถ่าน)
```
เลื่อนแปรงไปตามแกนนิวทรัลใหม่
```

#### ข้อเสีย
- ต้องปรับทุกครั้งที่โหลดเปลี่ยน
- ไม่นิยมใช้แล้ว
- ใช้เฉพาะเครื่องเก่า

## ความรุนแรง

### ขึ้นกับ
- กระแสอาร์มาเจอร์ (Ia)
- ความเร็วรอบ (N)
- ขนาดของเครื่อง

### มากที่สุดเมื่อ
- โหลดเต็ม (Ia สูง)
- ความเร็วสูง
- เครื่องขนาดใหญ่
