# RCD - Residual Current Device

## คำนิยาม
**RCD (Residual Current Device)** หรือ **เครื่องตัดไฟรั่ว** คืออุปกรณ์ป้องกันไฟฟ้ารั่วที่อาจทำให้เกิดไฟดูดหรือไฟช็อต

## ชื่อเรียกอื่น
- **RCCB** (Residual Current Circuit Breaker)
- **ELCB** (Earth Leakage Circuit Breaker) - ชื่อเก่า
- **GFI/GFCI** (Ground Fault Interrupter) - USA

## หลักการทำงาน

### ปกติ
```
กระแสเข้า (Line) = กระแสออก (Neutral)
I_in = I_out
ฟลักซ์รวมใน CT = 0
→ ไม่ตัด
```

### เมื่อมีไฟรั่ว
```
กระแสเข้า ≠ กระแสออก
I_in - I_out = I_leakage
ฟลักซ์รวมใน CT ≠ 0
→ เหนี่ยวนำ EMF
→ ตัดวงจร
```

## ประเภท

### ตามกระแสรั่ว (IΔn)
- **10 mA**: ป้องกันคนโดยตรง (เต้ารับห้องน้ำ)
- **30 mA**: ป้องกันคนทั่วไป (มาตรฐาน)
- **100 mA**: ป้องกันไฟไหม้
- **300 mA**: ป้องกันอุปกรณ์

### ตามประเภทกระแส
- **Type AC**: ตรวจจับ AC เท่านั้น
- **Type A**: ตรวจจับ AC + DC pulsating
- **Type B**: ตรวจจับทุกชนิด (รวม DC)

### ตามเวลาตัด
- **Instantaneous**: ตัดทันที (< 40 ms)
- **Time Delay (S-Type)**: หน่วงเวลา (ใช้ที่เมน)

## การติดตั้ง

### ตำแหน่งบังคับ (ตาม วสท.)
1. **เครื่องทำน้ำอุ่น** (Water Heater)
2. **เต้ารับในห้องน้ำ**
3. **เต้ารับใกล้อ่างล้างจาน** (< 1.5 m)
4. **เต้ารับกลางแจ้ง**
5. **ปลั๊กพ่วง** (Extension)

### การต่อ
```
Main Breaker → RCD → Sub Breakers
                |
            อุปกรณ์ที่ป้องกัน
```

### ข้อควรระวัง
- ต่อทั้ง Line และ Neutral ผ่าน RCD
- สาย Ground ไม่ผ่าน RCD
- ห้ามต่อ Neutral กับ Ground หลัง RCD

## การเลือกใช้

### 1. เลือกกระแสรั่ว
```
ป้องกันคน → 30 mA
ป้องกันไฟไหม้ → 100-300 mA
```

### 2. เลือกกระแสพิกัด
```
ตามโหลด
เช่น 16A, 32A, 63A
```

### 3. เลือก Type
```
โหลดทั่วไป → Type AC
มี VSD, UPS → Type A
```

## การทดสอบ

### ปุ่ม Test
- กดทุกเดือน
- ต้องตัดทันที
- ถ้าไม่ตัด → เสีย ต้องเปลี่ยน

### เครื่องทดสอบ
- ทดสอบเวลาตัด
- ทดสอบกระแสรั่ว
- ต้องตัดภายใน 40 ms (ที่ IΔn)

## ปัญหาที่พบบ่อย

### 1. ตัดบ่อย (Nuisance Tripping)
**สาเหตุ:**
- มีกระแสรั่วจริง
- โหลดมี Leakage สูง (ปกติ)
- สายไฟเก่า ฉนวนเสื่อม

**แก้ไข:**
- ตรวจหาจุดรั่ว
- แยกวงจร
- ใช้ RCD ที่ IΔn สูงขึ้น

### 2. ไม่ตัด
**สาเหตุ:**
- RCD เสีย
- ต่อผิด (Neutral ไม่ผ่าน RCD)

**แก้ไข:**
- เปลี่ยน RCD
- ตรวจสอบการต่อ

### 3. ตัดช้า
**สาเหตุ:**
- RCD เสื่อม
- กระแสรั่วใกล้ IΔn

**แก้ไข:**
- เปลี่ยน RCD

## ข้อจำกัด

### ไม่ป้องกัน
- ไฟช็อตระหว่าง Line-Neutral
- โอเวอร์โหลด
- ลัดวงจร

### ต้องใช้ร่วมกับ
- [[Circuit-Breaker]] (MCB/MCCB)
- [[Overload-Relay]]

## ความแตกต่าง RCD vs RCBO

| | RCD | RCBO |
|---|---|---|
| ป้องกันไฟรั่ว | ✓ | ✓ |
| ป้องกันโอเวอร์โหลด | ✗ | ✓ |
| ป้องกันลัดวงจร | ✗ | ✓ |
| ราคา | ถูก | แพง |
