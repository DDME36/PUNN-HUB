# Armature Winding - ขดลวดอาร์มาเจอร์

## คำนิยาม
**ขดลวดอาร์มาเจอร์ (Armature Winding)** คือขดลวดที่พันบนส่วนหมุน (Rotor) ของ [[DC-Motor]] หรือ DC Generator ซึ่งเป็นส่วนที่มีกระแสไหลและเกิดแรงบิดหรือ EMF

## ประเภทการพัน

### 1. Lap Winding (การพันแบบแล็ป)
```
จำนวนวงจรขนาน (A) = จำนวนขั้ว (P)
A = P
```

#### คุณสมบัติ
- กระแสสูง แรงดันต่ำ
- เหมาะกับเครื่องจักรกำลังสูง แรงดันต่ำ
- ใช้กับ DC Generator ขนาดใหญ่

#### การต่อ
- ปลายขดลวดต่อกับ [[Commutator]] ที่อยู่ใกล้กัน
- Simple Lap, Wave Lap

### 2. Wave Winding (การพันแบบเวฟ)
```
จำนวนวงจรขนาน (A) = 2
A = 2 (เสมอ)
```

#### คุณสมบัติ
- แรงดันสูง กระแสต่ำ
- เหมาะกับเครื่องจักรแรงดันสูง
- ใช้แปรงถ่านน้อยกว่า

#### การต่อ
- ปลายขดลวดต่อข้าม [[Commutator]]
- Simple Wave, Series Wave

## สมการสำคัญ

### EMF ที่เกิด
$$E = \frac{P\Phi ZN}{60A}$$

- P = จำนวนขั้ว
- Φ = ฟลักซ์ต่อขั้ว (Wb)
- Z = จำนวนตัวนำทั้งหมด
- N = ความเร็วรอบ (rpm)
- A = จำนวนวงจรขนาน

### กระแสอาร์มาเจอร์
$$I_a = \frac{I_{total}}{A}$$

## Armature Reaction

### คำนิยาม
[[Armature-Reaction]] คือผลกระทบของสนามแม่เหล็กจากกระแสในอาร์มาเจอร์ที่รบกวนสนามแม่เหล็กหลัก

### ผลกระทบ
1. **Demagnetization**: ลดสนามแม่เหล็กหลัก
2. **Cross-magnetization**: บิดเบือนสนามแม่เหล็ก
3. **Shift Neutral Axis**: แกนนิวทรัลเลื่อน

### การแก้ไข
- **Interpole**: ขั้วแม่เหล็กเสริม
- **Compensating Winding**: ขดลวดชดเชย
- **Brush Shift**: เลื่อนแปรงถ่าน (วิธีเก่า)

## การออกแบบ

### พิจารณา
- จำนวนรอบต่อขั้ว
- ขนาดเส้นลวด (รองรับกระแส)
- ฉนวนทนความร้อน
- การระบายความร้อน

- Class A: 105°C
- Class B: 130°C
- Class F: 155°C (มาตรฐานโรงงาน)
- Class H: 180°C
