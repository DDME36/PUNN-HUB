# Mesh Current Analysis - การวิเคราะห์กระแสเมช

## คำนิยาม
**Mesh Current Analysis** หรือ **Loop Analysis** คือวิธีวิเคราะห์วงจรโดยใช้ [[Kirchhoffs-Laws|KVL]] ในวงปิด (Mesh) เพื่อหากระแสในแต่ละวงปิด

## หลักการ
- กำหนดกระแสวน (Loop Current) ในแต่ละ Mesh
- เขียนสมการ KVL ในแต่ละ Mesh
- แก้สมการหากระแส
- กระแสจริง = ผลรวมของกระแสวน

## ขั้นตอน

### 1. กำหนดกระแสวน
```
แต่ละ Mesh มีกระแสวน: I₁, I₂, I₃, ...
ทิศทาง: ตามเข็มนาฬิกา (CW) ทั้งหมด
```

### 2. เขียนสมการ KVL
```
เดินรอบ Mesh ตามเข็มนาฬิกา:
∑ V = 0

+ แรงดันเพิ่ม (แบตเตอรี่ +→-)
- แรงดันตก (ตัวต้านทาน)
```

### 3. คำนวณแรงดันตก
```
ตัวต้านทานในวงเดียว: V = I × R
ตัวต้านทานร่วม: V = (I₁ - I₂) × R
```

### 4. แก้สมการ
```
ได้กระแสวน I₁, I₂, I₃, ...
กระแสจริง = ผลรวมกระแสวนที่ผ่าน
```

## ตัวอย่างการใช้งาน

### ตัวอย่าง 1: วงจร 2 Mesh
```
วงจร:
V₁ = 12V, V₂ = 6V
R₁ = 2Ω, R₂ = 3Ω, R₃ = 4Ω

กำหนด: I₁ (Mesh ซ้าย), I₂ (Mesh ขวา)

Mesh 1 (KVL):
V₁ - I₁R₁ - (I₁ - I₂)R₃ = 0
12 - 2I₁ - 4(I₁ - I₂) = 0
12 - 6I₁ + 4I₂ = 0

Mesh 2 (KVL):
-(I₂ - I₁)R₃ - I₂R₂ - V₂ = 0
-4(I₂ - I₁) - 3I₂ - 6 = 0
4I₁ - 7I₂ - 6 = 0

แก้สมการ:
I₁ = 2.4A
I₂ = 0.8A

กระแสผ่าน R₃ = I₁ - I₂ = 1.6A
```

### ตัวอย่าง 2: วงจร 3 Mesh
```
วงจร:
V_s = 24V
R₁ = R₂ = R₃ = R₄ = R₅ = 6Ω

กำหนด: I₁, I₂, I₃

Mesh 1:
V_s - I₁R₁ - (I₁ - I₂)R₄ = 0

Mesh 2:
-(I₂ - I₁)R₄ - I₂R₂ - (I₂ - I₃)R₅ = 0

Mesh 3:
-(I₃ - I₂)R₅ - I₃R₃ = 0

แก้สมการ 3 ตัวแปร
```

## เทคนิคพิเศษ

### Super Mesh
เมื่อมีแหล่งจ่ายกระแสระหว่าง 2 Mesh

```
I₁ - I₂ = I_s (ทราบแล้ว)

เขียน KVL รอบ Super Mesh:
∑ V = 0 (ของทั้ง 2 Mesh)
```

### แหล่งจ่ายแรงดัน
```
ง่าย! เขียน V_s ลงในสมการ KVL ตรงๆ
```

## การใช้กับ AC Circuit

### วงจร AC
```
ใช้ [[Complex-Numbers-AC]]
Ṽ = V∠θ
Z̃ = R + jX

KVL: ∑ Ṽ = 0
Ṽ = Ĩ × Z̃
```

### ตัวอย่าง AC
```
Ṽ_s = 220∠0° V
Z₁ = 3 + j4 Ω
Z₂ = 5 - j2 Ω

Mesh 1:
Ṽ_s - Ĩ₁Z₁ - (Ĩ₁ - Ĩ₂)Z₃ = 0

แก้สมการ Complex
```

## ข้อผิดพลาดที่พบบ่อย

### ❌ ทิศทางไม่สม่ำเสมอ
```
ผิด: I₁ CW, I₂ CCW (สับสน!)
ถูก: ทุก Mesh ใช้ CW หรือ CCW เหมือนกัน
```

### ❌ เครื่องหมายผิด
```
ผิด: V_s + I₁R₁ + I₂R₂ = 0
ถูก: V_s - I₁R₁ - I₂R₂ = 0
     (แรงดันตกเป็นลบ)
```

### ❌ ตัวต้านทานร่วม
```
ผิด: V = I₁ × R (ลืม I₂)
ถูก: V = (I₁ - I₂) × R

ถ้า I₁ และ I₂ ไปทิศทางเดียวกัน: (I₁ + I₂)
ถ้าทิศทางตรงข้าม: (I₁ - I₂)
```

### ❌ แหล่งจ่ายกระแส
```
ผิด: เขียน KVL ผ่าน I_s
ถูก: ใช้ Super Mesh

หรือ

แปลง I_s → V_s (ถ้าทำได้)
```

## เปรียบเทียบวิธีการ

| วิธี | ใช้กฎ | ตัวแปร | จำนวนสมการ | เหมาะกับ |
|---|---|---|---|---|
| Mesh | KVL | กระแส | = จำนวน Mesh | แหล่งจ่ายแรงดัน |
| [[Node-Voltage-Analysis\|Node]] | KCL | แรงดัน | = จำนวนจุดต่อ - 1 | แหล่งจ่ายกระแส |

## การเลือกวิธี

### ใช้ Mesh เมื่อ:
- มีแหล่งจ่ายแรงดันเยอะ
- วงจรเป็น Planar
- จำนวน Mesh < จำนวนจุดต่อ

### ใช้ Node เมื่อ:
- มีแหล่งจ่ายกระแสเยอะ
- วงจรไม่เป็น Planar
- จำนวนจุดต่อ < จำนวน Mesh

## ตัวอย่างโจทย์ยาก

### โจทย์: วงจร Bridge
```
วงจร Wheatstone Bridge:
V_s = 12V
R₁ = 2Ω, R₂ = 4Ω
R₃ = 3Ω, R₄ = 6Ω
R₅ = 5Ω (ตรงกลาง)

หา: I₁, I₂, I₃ (3 Mesh)

วิธีทำ:
Mesh 1 (ซ้าย):
V_s - I₁R₁ - (I₁ - I₃)R₅ - (I₁ - I₂)R₃ = 0

Mesh 2 (ขวา):
-(I₂ - I₁)R₃ - I₂R₂ - (I₂ - I₃)R₄ = 0

Mesh 3 (กลาง):
-(I₃ - I₁)R₅ - (I₃ - I₂)R₄ = 0

แก้สมการ 3 ตัวแปร:
I₁ = 2.1A
I₂ = 0.9A
I₃ = 1.5A

กระแสผ่าน R₅ = I₁ - I₃ = 0.6A
```

## Tips & Tricks

### 1. ทิศทางสม่ำเสมอ
- ใช้ CW ทุก Mesh
- ลดความสับสน

### 2. ตรวจสอบ Planar
- วาดวงจรใหม่
- ถ้าสายไม่ตัดกัน → Planar

### 3. ตรวจสอบคำตอบ
- ใช้ KCL ที่จุดต่อตรวจสอบ
- ใช้ [[Ohms-Law]] ตรวจสอบแรงดัน

### 4. Matrix Form
```
[R₁₁  R₁₂  R₁₃] [I₁]   [V₁]
[R₂₁  R₂₂  R₂₃] [I₂] = [V₂]
[R₃₁  R₃₂  R₃₃] [I₃]   [V₃]

ใช้ Matrix แก้สมการ (ง่ายกว่า)
```

## การประยุกต์ใช้

### 1. วงจรซับซ้อน
- วิเคราะห์วงจรที่มีหลาย Loop
- หากระแสในแต่ละสาขา

### 2. Filter Circuit
- วิเคราะห์วงจรกรอง
- หา Transfer Function

### 3. Transformer Circuit
- วิเคราะห์วงจร [[Transformer]]
- หากระแสปฐมภูมิและทุติยภูมิ
