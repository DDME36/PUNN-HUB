# VSD - Variable Speed Drive

## คำนิยาม
**VSD (Variable Speed Drive)** หรือ **VFD (Variable Frequency Drive)** คืออุปกรณ์ควบคุมความเร็ว [[Motor]] โดยการปรับความถี่และแรงดันไฟฟ้า

## ชื่อเรียกอื่น
- **VFD** (Variable Frequency Drive)
- **Inverter** (อินเวอร์เตอร์)
- **AC Drive**
- **Frequency Converter**

## หลักการทำงาน

### ขั้นตอน
```
AC Input → Rectifier → DC Bus → Inverter → AC Output
(380V,50Hz)  (AC→DC)   (Filter)  (DC→AC)  (0-50Hz)
```

### 1. Rectifier (เรียงกระแส)
- แปลง AC → DC
- ใช้ Diode Bridge

### 2. DC Bus (บัสกระแสตรง)
- เก็บพลังงาน DC
- มี Capacitor กรอง

### 3. Inverter (อินเวอร์เตอร์)
- แปลง DC → AC
- ใช้ IGBT สวิตช์
- ปรับความถี่และแรงดัน

## การควบคุมความเร็ว

### ความสัมพันธ์
$$N = \frac{120f}{P}$$

- เพิ่ม f → เพิ่ม N
- ลด f → ลด N

### V/f Control
$$\frac{V}{f} = \text{constant}$$

- รักษาอัตราส่วน V/f คงที่
- ป้องกันแกนเหล็กอิ่มตัว
- วิธีที่ใช้กันมากที่สุด

### ตัวอย่าง
```
ปกติ: 380V, 50Hz → V/f = 7.6
ครึ่งความเร็ว: 190V, 25Hz → V/f = 7.6
```

## ข้อดี

### 1. ประหยัดพลังงาน
```
พัดลม/ปั๊ม:
P ∝ N³

ลดความเร็ว 20% → ประหยัด 50%!
```

### 2. สตาร์ทนุ่มนวล
- กระแสสตาร์ทต่ำ (1-1.5× In)
- ไม่มีกระแสกระชาก
- ไม่ทำให้แรงดันตก

### 3. ควบคุมแม่นยำ
- ปรับความเร็วได้ละเอียด
- ตอบสนองเร็ว
- ควบคุมแรงบิดได้

### 4. ป้องกันในตัว
- Overload
- Overvoltage/Undervoltage
- Phase Loss
- Overtemperature

## ข้อเสีย

### 1. ราคาแพง
- แพงกว่า [[Star-Delta-Starter]] มาก
- แต่คุ้มค่าถ้าใช้ปรับความเร็วบ่อย

### 2. Harmonics
- สร้าง [[Harmonics]] สูง
- รบกวนอุปกรณ์อื่น
- ต้องใช้ Filter

### 3. Motor Heating
- PWM ทำให้มอเตอร์ร้อนขึ้น
- ต้องใช้มอเตอร์ Inverter Duty

### 4. EMI (Electromagnetic Interference)
- สัญญาณรบกวน
- ต้องใช้สาย Shielded
- ต้องต่อลงดินดี

## การติดตั้ง

### ตำแหน่ง
- ติดตั้งในตู้ควบคุม
- ระบายความร้อนดี
- ห่างจากอุปกรณ์ที่ไวต่อ EMI

### สายไฟ
- **Input**: สายปกติ
- **Output**: สาย Shielded
- ความยาวไม่เกิน 50-100 m

### การต่อลงดิน
- ต่อลงดินดี (< 5Ω)
- ต่อ Shield ทั้งสองข้าง
- ใช้สาย Ground ขนาดใหญ่

## การตั้งค่า

### พารามิเตอร์สำคัญ
1. **Motor Rating**: กำลัง, แรงดัน, กระแส
2. **Acceleration Time**: เวลาเร่ง (5-20 s)
3. **Deceleration Time**: เวลาชะลอ (5-20 s)
4. **Max Frequency**: ความถี่สูงสุด (50-60 Hz)
5. **V/f Curve**: เส้นโค้ง V/f

### การปรับแต่ง
- ทดลองปรับ Acceleration/Deceleration
- ปรับ V/f ถ้ามอเตอร์ร้อน
- ตั้ง Protection Parameters

## การใช้งาน

### เหมาะกับ
- **พัดลม**: ประหยัดพลังงานมาก
- **ปั๊มน้ำ**: ควบคุมอัตราการไหล
- **สายพาน**: ควบคุมความเร็ว
- **เครน**: ควบคุมแรงบิด

### ไม่เหมาะกับ
- โหลดที่ไม่ต้องปรับความเร็ว
- งบประมาณจำกัด
- สภาพแวดล้อมที่มี EMI ไวมาก

## ปัญหาที่พบบ่อย

### 1. Overcurrent
- โหลดหนักเกินไป
- Acceleration เร็วเกินไป
- แก้: เพิ่มเวลาเร่ง

### 2. Overvoltage
- Deceleration เร็วเกินไป
- มอเตอร์ป้อนไฟกลับ
- แก้: เพิ่มเวลาชะลอ, ใช้ Brake Resistor

### 3. Motor Overheating
- ใช้ความเร็วต่ำนานๆ
- ระบายความร้อนไม่ดี
- แก้: ใช้ External Fan
