# Hysteresis Loss - การสูญเสียฮิสเตอรีซิส

## คำนิยาม
**การสูญเสียฮิสเตอรีซิส (Hysteresis Loss)** คือการสูญเสียพลังงานในแกนเหล็กเนื่องจากการกลับทิศของสนามแม่เหล็กซ้ำๆ ทำให้โมเลกุลแม่เหล็กในเหล็กต้องหมุนกลับไปมา

## สาเหตุ

### Magnetic Domain
- เหล็กประกอบด้วย Magnetic Domain (โดเมนแม่เหล็ก)
- เมื่อมีสนามแม่เหล็กภายนอก โดเมนจะหมุนตัวเรียงตัว
- เมื่อสนามกลับทิศ โดเมนต้องหมุนกลับ
- การหมุนนี้ต้องใช้พลังงาน → สูญเสียเป็นความร้อน

### Hysteresis Loop
```
กราฟ B-H (ฟลักซ์-สนามแม่เหล็ก)
พื้นที่ใน Loop = พลังงานสูญเสียต่อรอบ
```

## สูตร Steinmetz
$$P_h = K_h f B_{max}^n V$$

- Ph = การสูญเสียฮิสเตอรีซิส (W)
- Kh = ค่าคงที่ของวัสดุ
- f = ความถี่ (Hz)
- Bmax = ความหนาแน่นฟลักซ์สูงสุด (T)
- n = ค่าเลขชี้กำลัง (1.5-2.5)
- V = ปริมาตรแกน (m³)

## ความสัมพันธ์

### ขึ้นกับความถี่
$$P_h \propto f$$
- ความถี่สูง → สูญเสียมาก
- ไฟ 60 Hz สูญเสียมากกว่า 50 Hz

### ขึ้นกับฟลักซ์
$$P_h \propto B_{max}^{1.6-2.0}$$
- ฟลักซ์สูง → สูญเสียมากขึ้นแบบยกกำลัง

## การลดการสูญเสีย

### 1. เลือกวัสดุที่เหมาะสม
- **Silicon Steel**: ผสม Si 3-4% → ลด Ph ได้ 50%
- **Grain Oriented Steel**: เรียงเกรนตามทิศทาง
- **Amorphous Metal**: Ph ต่ำมาก แต่แพง

### 2. Annealing (อบอ่อน)
- อบเหล็กที่อุณหภูมิสูง
- ลด Internal Stress
- ทำให้โดเมนหมุนง่ายขึ้น

### 3. ลดฟลักซ์
- ออกแบบให้ Bmax ต่ำ
- ใช้แกนใหญ่ขึ้น

## ในเครื่องจักรไฟฟ้า

### Transformer
- สูญเสียตลอดเวลา (แม้ไม่มีโหลด)
- เรียกว่า No-Load Loss
- ประมาณ 20-30% ของ Core Loss

### Motor
- เกิดทั้งใน Stator และ Rotor
- ใน Rotor ความถี่ต่ำกว่า (= Slip × f)

## ความแตกต่างจาก Eddy Current Loss
