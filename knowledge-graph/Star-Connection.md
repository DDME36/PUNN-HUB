# Star Connection (Y) - การต่อแบบสตาร์

## คำนิยาม
**การต่อแบบสตาร์ (Star Connection หรือ Wye Connection)** คือการต่อขดลวด 3 เฟสโดยรวมปลายด้านหนึ่งเข้าด้วยกันเป็นจุด Neutral

## สัญลักษณ์
```
     A
     |
N----+----B
     |
     C
```

## สูตรหลัก

### แรงดัน
$$V_{line} = \sqrt{3} \times V_{phase}$$
$$V_L = 1.732 \times V_P$$

### กระแส
$$I_{line} = I_{phase}$$
$$I_L = I_P$$

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ ระบบ 3 เฟส
- [[Star-Connection]] → ใช้ใน [[Three-Phase-System]]
- [[Star-Connection]] → ใช้ [[Square-Root-3-Rule]]
- [[Star-Connection]] vs [[Delta-Connection]]

### 🔌 การใช้งาน
- [[Transformer]] ฝั่งแรงดันสูง
- [[Motor]] ตอนสตาร์ท
- ระบบจำหน่าย 380V/220V

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หาแรงดันสาย
```
V_phase = 220V (เฟสกับนิวทรัล)
V_line = √3 × V_phase
V_line = 1.732 × 220 = 381V ≈ 380V
```

### ตัวอย่าง 2: หากระแส
```
S = 100 kVA, V_L = 380V
I_L = S / (√3 × V_L)
I_L = 100,000 / (1.732 × 380)
I_L = 152A

I_phase = I_L = 152A (เท่ากัน!)
```

### ตัวอย่าง 3: หากำลัง
```
V_L = 380V, I_L = 100A, PF = 0.85
P = √3 × V_L × I_L × cosθ
P = 1.732 × 380 × 100 × 0.85
P = 55,944 W ≈ 56 kW
```

## ข้อดี
- มีจุด Neutral → ดึงไฟ 1 เฟสได้
- แรงดันเฟสต่ำกว่า → ฉนวนน้อยกว่า
- เหมาะกับแรงดันสูง

## ข้อเสีย
- กระแสสายเท่ากับกระแสเฟส
- ถ้าโหลดไม่สมดุล → มีกระแสใน N

## การใช้งาน

### ระบบจำหน่าย
```
380V/220V (3-Phase/1-Phase)
- ระหว่างเฟส (A-B, B-C, C-A) = 380V
- เฟสกับนิวทรัล (A-N, B-N, C-N) = 220V
```

### Star-Delta Starter
```
สตาร์ท: Star (กระแสต่ำ)
ทำงาน: Delta (แรงบิดสูง)
```
