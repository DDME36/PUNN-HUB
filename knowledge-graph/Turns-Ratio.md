# Turns Ratio (a) - อัตราส่วนจำนวนรอบ

## คำนิยาม
**อัตราส่วนจำนวนรอบ (Turns Ratio)** คืออัตราส่วนระหว่างจำนวนรอบขดลวดปฐมภูมิและทุติยภูมิของ [[Transformer]]

## สัญลักษณ์
- **a** = Turns Ratio
- **N1** = จำนวนรอบปฐมภูมิ
- **N2** = จำนวนรอบทุติยภูมิ

## สูตร
$$a = \frac{N_1}{N_2} = \frac{V_1}{V_2} = \frac{I_2}{I_1}$$

## ความสัมพันธ์

### แรงดัน (สัดส่วนตรง)
$$\frac{V_1}{V_2} = \frac{N_1}{N_2} = a$$

### กระแส (สัดส่วนกลับ!)
$$\frac{I_1}{I_2} = \frac{N_2}{N_1} = \frac{1}{a}$$

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: Step-down
```
N1 = 1,000 รอบ, N2 = 100 รอบ
a = N1/N2 = 1,000/100 = 10

V1 = 22,000V
V2 = V1/a = 22,000/10 = 2,200V
```

### ตัวอย่าง 2: หากระแส
```
a = 10, I2 = 100A
I1 = I2/a = 100/10 = 10A
```

## ประเภท
- **a > 1**: Step-down (ลดแรงดัน)
- **a < 1**: Step-up (เพิ่มแรงดัน)
- **a = 1**: Isolation (แยกวงจร)
