# Motor - มอเตอร์

## คำนิยาม
**มอเตอร์ (Motor)** คืออุปกรณ์ที่แปลงพลังงานไฟฟ้าเป็นพลังงานกล ใช้หลักการสนามแม่เหล็กหมุน

## ประเภทหลัก
1. **AC Motor**: Induction Motor (ใช้มากที่สุด)
2. **[[DC-Motor]]**: DC Brush, Brushless
3. **Synchronous Motor**: ความเร็วคงที่

## สัญลักษณ์วงจร
```
   ⊕
  /|\
 / | \
   M
```

## สูตรพื้นฐาน

### ความเร็วซิงโครนัส ([[Synchronous-Speed]])
$$N_s = \frac{120f}{P}$$
- f = ความถี่ (Hz)
- P = จำนวนขั้ว (poles)

### สลิป ([[Slip]])
$$s = \frac{N_s - N_r}{N_s}$$
- $N_s$ = ความเร็วซิงโครนัส
- $N_r$ = ความเร็วโรเตอร์จริง

### ความเร็วโรเตอร์
$$N_r = N_s(1 - s)$$

### แรงบิด (Torque)
$$T = \frac{P \times 9550}{N}$$
- P = กำลัง (kW)
- N = ความเร็ว (rpm)
- T = แรงบิด (N·m)

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ พื้นฐาน
- [[Motor]] → ใช้ [[Three-Phase-System]]
- [[Motor]] → มี [[Inductor-L]] (ขดลวด)
- [[Motor]] → สร้าง [[Reactive-Power-Q]] (ล้าหลัง)

### 🔺 กำลังและ PF
- [[Motor]] → [[Power-Factor-PF]] ต่ำ (0.7-0.9)
- [[Motor]] → ต้องการ [[Power-Factor-Correction]]
- [[Motor]] → ใช้ [[Capacitor-Bank]] แก้ PF

### ⚙️ การต่อ
- [[Star-Connection]]: สตาร์ท (กระแสต่ำ)
- [[Delta-Connection]]: ทำงาน (แรงบิดสูง)
- [[Star-Delta-Starter]]: ลดกระแสสตาร์ท

## เนมเพลต ([[Nameplate-Reading]])
```
10 HP (7.5 kW)
380V, 3-Phase, 50Hz
15A, 1450 rpm
PF = 0.85, Eff = 90%
Class F
```

### การอ่าน
- **10 HP (7.5 kW)**: กำลังขาออกที่เพลา (Output Power)
- **380V**: แรงดันสาย (Line Voltage)
- **15A**: กระแสสาย (Line Current)
- **1450 rpm**: ความเร็วโหลดเต็ม
- **PF = 0.85**: ตัวประกอบกำลัง
- **Eff = 90%**: ประสิทธิภาพ
- **Class F**: ฉนวนทน 155°C

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หาความเร็วซิงโครนัส
```
f = 50 Hz, P = 4 poles
N_s = 120f/P = 120×50/4 = 1500 rpm
```

### ตัวอย่าง 2: หาสลิป
```
N_s = 1500 rpm, N_r = 1450 rpm
s = (N_s - N_r)/N_s = (1500-1450)/1500
s = 50/1500 = 0.0333 = 3.33%
```

### ตัวอย่าง 3: หากำลังไฟฟ้าที่กิน
```
P_out = 7.5 kW, Eff = 90%
P_in = P_out / Eff = 7.5 / 0.9 = 8.33 kW
```

### ตัวอย่าง 4: หากระแสจากเนมเพลต
```
P = 7.5 kW, V = 380V, PF = 0.85
I = P / (√3 × V × PF)
I = 7,500 / (1.732 × 380 × 0.85)
I = 13.4A
```

### ตัวอย่าง 5: แปลง rpm → rad/s
```
N = 1450 rpm
ω = N × 2π / 60
ω = 1450 × 2π / 60 = 151.8 rad/s
```

## คุณสมบัติการทำงาน

### กระแสสตาร์ท
- **กระแสสตาร์ท**: 5-7 เท่าของกระแสทำงาน
- **ปัญหา**: แรงดันตก, เบรกเกอร์ตก
- **แก้ไข**: ใช้ [[Star-Delta-Starter]]

### Power Factor
- **โหลดเต็ม**: PF = 0.85-0.90
- **โหลดต่ำ**: PF = 0.3-0.5 (แย่มาก!)
- **แก้ไข**: ใช้ [[Capacitor-Bank]]

### ประสิทธิภาพ
- **โหลดเต็ม**: 85-95%
- **โหลด 75%**: สูงสุด
- **โหลดต่ำ**: ลดลง

## ข้อผิดพลาดที่พบบ่อย

### ❌ ความเร็วซิงโครนัส
```
ผิด: ใช้ N_r (ความเร็วจริง) ในสูตร N_s = 120f/P
ถูก: N_s คือความเร็วสนามแม่เหล็ก (ไม่ใช่โรเตอร์)
     N_r < N_s เสมอ (มีสลิป)
```

### ❌ กำลังบนเนมเพลต
```
ผิด: คิดว่า 10 HP คือกำลังไฟฟ้าที่กิน
ถูก: 10 HP คือกำลังกลที่เพลา (Output)
     กำลังไฟฟ้า = Output / Efficiency
```

### ❌ การแปลงหน่วย
```
ผิด: ω = N × 2π (ลืมหาร 60)
ถูก: ω = N × 2π / 60 (rpm → rad/s)
```

### ❌ PF ต่ำ
```
ผิด: คิดว่ามอเตอร์พัง
ถูก: มอเตอร์ต้องการ Q สูง (ปกติ)
     แก้ด้วย Capacitor Bank
```

## การแก้ PF ด้วย Capacitor Bank
```
มอเตอร์ 10 kW, PF = 0.7
ต้องการ PF = 0.95

ขั้นตอน:
1. Q₁ = P × tan(cos⁻¹(0.7)) = 10 × 1.02 = 10.2 kVAR
2. Q₂ = P × tan(cos⁻¹(0.95)) = 10 × 0.33 = 3.3 kVAR
3. Q_C = Q₁ - Q₂ = 10.2 - 3.3 = 6.9 kVAR

ติดตั้ง Capacitor Bank 7 kVAR
```

## สภาวะการทำงาน
- **$N_r < N_s$**: สภาวะมอเตอร์ (กินไฟ → หมุน)
- **$N_r = N_s$**: Synchronous Motor
- **$N_r > N_s$**: สภาวะเครื่องกำเนิด (หมุน → ผลิตไฟ)
