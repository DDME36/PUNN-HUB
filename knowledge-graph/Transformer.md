# Transformer - หม้อแปลงไฟฟ้า

## คำนิยาม
**หม้อแปลงไฟฟ้า (Transformer)** คืออุปกรณ์ที่ใช้เปลี่ยนระดับแรงดันไฟฟ้า AC โดยใช้หลักการเหนี่ยวนำแม่เหล็ก

## หลักการทำงาน
- ขดลวดปฐมภูมิ (Primary) สร้างสนามแม่เหล็ก
- สนามแม่เหล็กผ่านแกนเหล็ก
- เหนี่ยวนำให้เกิดแรงดันที่ขดลวดทุติยภูมิ (Secondary)

## สัญลักษณ์วงจร
```
Primary  ||  Secondary
  )))    ||    (((
  N₁     ||    N₂
```

## สูตรพื้นฐาน

### อัตราส่วนแรงดัน ([[Turns-Ratio]])
$$\frac{V_1}{V_2} = \frac{N_1}{N_2} = a$$

### อัตราส่วนกระแส (สัดส่วนกลับกัน!)
$$\frac{I_1}{I_2} = \frac{N_2}{N_1} = \frac{1}{a}$$

### กำลังไฟฟ้า (เท่ากัน - ในอุดมคติ)
$$S_1 = S_2$$
$$V_1 \times I_1 = V_2 \times I_2$$

### ⚙️ การใช้งาน
- สายส่ง: 500kV → 115kV → 22kV → 380V
- โรงงาน: 22kV → 380V
- บ้าน: 380V → 220V (ผ่าน Star connection)
- การวัด: [[CT-VT]] (หม้อแปลงวัด)

## ประเภทของหม้อแปลง

### 1. ตามจำนวนเฟส
- **1 เฟส**: บ้านเรือน
- **3 เฟส**: โรงงาน, สายส่ง

### 2. ตามหน้าที่
- **Step-up**: เพิ่มแรงดัน (N₂ > N₁)
- **Step-down**: ลดแรงดัน (N₂ < N₁)
- **Isolation**: แยกวงจร (N₁ = N₂)

### 3. ตามการต่อ
- **Star-Star** (Y-Y)
- **Delta-Delta** (Δ-Δ)
- **Star-Delta** (Y-Δ)
- **Delta-Star** (Δ-Y)

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หาแรงดันทุติยภูมิ
```
V₁ = 22,000V, N₁ = 1000 รอบ, N₂ = 100 รอบ
V₂ = V₁ × (N₂/N₁) = 22,000 × (100/1000)
V₂ = 2,200V
```

### ตัวอย่าง 2: หากระแสปฐมภูมิ
```
S = 100 kVA, V₁ = 22 kV
I₁ = S/V₁ = 100,000/22,000 = 4.55A
```

### ตัวอย่าง 3: หากระแสทุติยภูมิ
```
S = 100 kVA, V₂ = 380V
I₂ = S/V₂ = 100,000/380 = 263A
```

### ตัวอย่าง 4: ตรวจสอบ S₁ = S₂
```
S₁ = V₁ × I₁ = 22,000 × 4.55 = 100 kVA ✓
S₂ = V₂ × I₂ = 380 × 263 = 100 kVA ✓
```

## เนมเพลตหม้อแปลง
```
100 kVA
22kV / 380V
50 Hz
%Z = 4%
```

### การอ่าน
- **100 kVA**: กำลังปรากฏสูงสุด ([[Apparent-Power-S]])
- **22kV / 380V**: แรงดันปฐมภูมิ / ทุติยภูมิ
- **50 Hz**: ความถี่
- **%Z = 4%**: [[Percent-Impedance-Z]]

## ข้อผิดพลาดที่พบบ่อย (CRITICAL!)

### ❌ กับดัก PF
```
ผิด: I = S / V × PF
ถูก: I = S / V (ไม่ต้องคูณ PF!)

เหตุผล: S (kVA) บนเนมเพลตคือกำลังปรากฏ
PF เป็นของโหลด ไม่ใช่ของหม้อแปลง
```

### ❌ อัตราส่วนกระแส
```
ผิด: I₁/I₂ = N₁/N₂
ถูก: I₁/I₂ = N₂/N₁ (สัดส่วนกลับกัน!)
```

### ❌ สาย vs เฟส (Delta)
```
ผิด: I_coil = I_line (ใน Delta)
ถูก: I_coil = I_line / √3

ตัวอย่าง:
หม้อแปลง 100kVA, 380V, Delta
I_line = 100,000 / (√3 × 380) = 152A
I_coil = 152 / √3 = 87.7A
```

### ❌ การหาพิกัด
```
ผิด: คิดข้ามฝั่ง (ใช้ V₁ หา I₂)
ถูก: แยกคำนวณแต่ละฝั่ง
- ฝั่งปฐมภูมิ: I₁ = S / V₁
- ฝั่งทุติยภูมิ: I₂ = S / V₂
```

## การสูญเสีย
1. **Core Loss**: สูญเสียในแกนเหล็ก ([[Hysteresis-Loss]] + [[Eddy-Current-Loss]])
2. **Copper Loss**: $I^2R$ ในขดลวด
3. **ประสิทธิภาพ**: 95-99%

## ประสิทธิภาพ
$$\eta = \frac{P_{out}}{P_{in}} = \frac{P_{out}}{P_{out} + P_{loss}}$$

## การทดสอบ
1. **Open Circuit Test**: หา Core Loss
2. **Short Circuit Test**: หา Copper Loss และ %Z
