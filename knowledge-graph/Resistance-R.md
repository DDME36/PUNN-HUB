# Resistance (R) - ความต้านทาน

## คำนิยาม
**ความต้านทาน (Resistance)** คือคุณสมบัติของวัสดุที่ขัดขวางการไหลของกระแสไฟฟ้า ทำให้เกิดการสูญเสียพลังงานเป็นความร้อน

## สัญลักษณ์
- **R** = Resistance (โอห์ม)
- หน่วย: **Ohm (Ω)**

## สูตรพื้นฐาน
$$R = \frac{V}{I}$$
(จาก [[Ohms-Law]])

$$R = \rho \frac{L}{A}$$
- ρ = ความต้านทานจำเพาะของวัสดุ
- L = ความยาวตัวนำ
- A = พื้นที่หน้าตัด

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ แนวคิดพื้นฐาน
- [[Voltage-V]] ÷ [[Current-I]] → [[Resistance-R]]
- [[Resistance-R]] → ส่วนจริงของ [[Impedance-Z]]
- [[Resistance-R]] → ทำให้เกิด [[Real-Power-P]]

### 🔌 การต่อวงจร
- [[Series-Connection]]: $R_{total} = R_1 + R_2 + R_3 + ...$
- [[Parallel-Connection]]: $\frac{1}{R_{total}} = \frac{1}{R_1} + \frac{1}{R_2} + ...$

### 🌡️ ผลกระทบจากอุณหภูมิ
$$R_t = R_0[1 + \alpha(T - T_0)]$$
- α = สัมประสิทธิ์อุณหภูมิ
- ทองแดง: α ≈ 0.00393/°C

## ตัวอย่างค่าความต้านทาน
- **ทองแดง**: ρ = 1.68 × 10⁻⁸ Ω·m (ตัวนำดี)
- **อลูมิเนียม**: ρ = 2.82 × 10⁻⁸ Ω·m
- **ยาง**: ρ > 10¹³ Ω·m (ฉนวน)

## ข้อผิดพลาดที่พบบ่อย
❌ **ผิด:** ความต้านทานไม่เปลี่ยนตามอุณหภูมิ  
✅ **ถูก:** ความต้านทานเพิ่มขึ้นเมื่ออุณหภูมิสูงขึ้น

❌ **ผิด:** สายไฟยาว 2 เท่า ความต้านทานเพิ่ม 2 เท่า  
✅ **ถูก:** ถูกต้อง! (R ∝ L)

❌ **ผิด:** ใน AC ใช้แค่ R  
✅ **ถูก:** ต้องใช้ [[Impedance-Z]] = R + jX

## การสูญเสียพลังงาน
$$P_{loss} = I^2 R$$
(นี่คือเหตุผลที่สายส่งแรงสูงใช้แรงดันสูงเพื่อลดกระแส)
