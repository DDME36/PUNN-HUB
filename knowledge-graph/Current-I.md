# Current (I) - กระแสไฟฟ้า

## คำนิยาม
**กระแสไฟฟ้า (Current)** คืออัตราการไหลของประจุไฟฟ้าผ่านตัวนำในหนึ่งหน่วยเวลา

## สัญลักษณ์
- **I** = Current (แอมแปร์)
- หน่วย: **Ampere (A)**

## สูตรพื้นฐาน
$$I = \frac{V}{R}$$
(จาก [[Ohms-Law]])

$$I = \frac{P}{V}$$
(จาก [[Electrical-Power-P]])

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ แนวคิดพื้นฐาน
- [[Voltage-V]] → ผลักดัน → [[Current-I]]
- [[Current-I]] × [[Resistance-R]] → [[Voltage-V]]
- [[Current-I]]² × [[Resistance-R]] → การสูญเสียพลังงาน

### 🔌 ในวงจร
- [[Series-Connection]]: กระแสเท่ากันทุกจุด
- [[Parallel-Connection]]: กระแสแยกสาขา

### 🏭 ในระบบ 3 เฟส
- [[Delta-Connection]]: $I_{line} = \sqrt{3} \times I_{phase}$
- [[Star-Connection]]: $I_{line} = I_{phase}$

### ⚙️ ในเครื่องจักร
- [[Transformer]]: $\frac{I_1}{I_2} = \frac{N_2}{N_1}$ (สัดส่วนกลับกับ [[Turns-Ratio]])
- [[Motor]]: กระแสสตาร์ทสูงกว่ากระแสทำงานปกติ 5-7 เท่า

## ประเภทของกระแส
1. **DC (Direct Current)**: ไหลทิศทางเดียว
2. **AC (Alternating Current)**: สลับทิศทาง 50/60 Hz

## ข้อผิดพลาดที่พบบ่อย
❌ **ผิด:** กระแสไหลจากบวกไปลบ  
✅ **ถูก:** อิเล็กตรอนไหลจากลบไปบวก (แต่เราตกลงใช้ทิศกระแสจากบวกไปลบ)

❌ **ผิด:** ในวงจรอนุกรม กระแสแยกสาขา  
✅ **ถูก:** ใน [[Series-Connection]] กระแสเท่ากันทุกจุด

## อันตรายจากกระแส
- **1-5 mA**: รู้สึกได้
- **10-20 mA**: กล้ามเนื้อกระตุก
- **50 mA**: อันตรายถึงชีวิต
- **> 100 mA**: หัวใจหยุดเต้น
