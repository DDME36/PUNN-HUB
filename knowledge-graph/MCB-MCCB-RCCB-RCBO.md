# MCB, MCCB, RCCB, RCBO - อุปกรณ์ป้องกันวงจรไฟฟ้า

## ภาพรวม

```
┌─────────────────────────────────────────────────┐
│  อุปกรณ์ป้องกันวงจรไฟฟ้า                        │
├─────────────────────────────────────────────────┤
│                                                 │
│  MCB    → ป้องกัน Overload + Short Circuit    │
│  MCCB   → MCB ขนาดใหญ่ (ปรับได้)               │
│  RCCB   → ป้องกันไฟรั่ว (Residual Current)     │
│  RCBO   → MCB + RCCB รวมกัน                    │
│  ELCB   → เก่า (ใช้ RCCB แทน)                  │
│                                                 │
└─────────────────────────────────────────────────┘
```

## 1. MCB (Miniature Circuit Breaker)

### คำนิยาม
**MCB** คือเบรกเกอร์ขนาดเล็ก ป้องกันโอเวอร์โหลดและลัดวงจร ใช้ในบ้านและอาคาร

### สัญลักษณ์
```
  ─┤├─
   MCB
```

### ขนาดที่ใช้บ่อย
- **6A, 10A, 16A**: หลอดไฟ, ปลั๊ก
- **20A, 32A**: แอร์, เครื่องทำน้ำอุ่น
- **40A, 50A, 63A**: เมนบ้าน

### Trip Curve (เส้นโค้งการตัด)

#### Type B (3-5 × In)
- ใช้กับ: หลอดไฟ, ปลั๊กทั่วไป
- ตัดที่: 3-5 เท่าของกระแสเรต

#### Type C (5-10 × In)
- ใช้กับ: ทั่วไป, แอร์, ตู้เย็น
- ตัดที่: 5-10 เท่าของกระแสเรต
- **ใช้มากที่สุดในบ้าน**

#### Type D (10-20 × In)
- ใช้กับ: มอเตอร์, หม้อแปลง
- ตัดที่: 10-20 เท่าของกระแสเรต

### ตัวอย่างการเลือก MCB

#### ตัวอย่าง 1: แอร์ 9,000 BTU
```
กำลัง: 900W
แรงดัน: 220V
กระแส: I = P/V = 900/220 = 4.1A

เลือก MCB: 10A Type C
(1.25 × 4.1 = 5.1A → ใช้ 10A)
```

#### ตัวอย่าง 2: เครื่องทำน้ำอุ่น
```
กำลัง: 3,500W
กระแส: 3,500/220 = 15.9A

เลือก MCB: 20A Type C
```

#### ตัวอย่าง 3: เมนบ้าน
```
โหลดรวม: 30A
เลือก MCB: 40A หรือ 50A Type C
```

### โครงสร้าง MCB
```
┌─────────────────┐
│   Terminal      │ ← ขั้วต่อ
├─────────────────┤
│  Bimetal Strip  │ ← ป้องกัน Overload (ช้า)
├─────────────────┤
│  Magnetic Coil  │ ← ป้องกัน Short Circuit (เร็ว)
├─────────────────┤
│  Arc Chamber    │ ← ดับประกายไฟ
├─────────────────┤
│   Contacts      │ ← จุดสัมผัส
└─────────────────┘
```

### ข้อดี-ข้อเสีย

#### ข้อดี
- ราคาถูก
- ติดตั้งง่าย
- Reset ได้ (ไม่ต้องเปลี่ยน)
- ปลอดภัย

#### ข้อเสีย
- ไม่ป้องกันไฟรั่ว
- Breaking Capacity จำกัด (6-10 kA)

## 2. MCCB (Molded Case Circuit Breaker)

### คำนิยาม
**MCCB** คือเบรกเกอร์ขนาดใหญ่ ปรับค่าได้ ใช้ในโรงงานและอาคารขนาดใหญ่

### สัญลักษณ์
```
  ═╪═
  MCCB
```

### ขนาดที่ใช้บ่อย
- **50A-100A**: เมนย่อย
- **100A-250A**: เมนหลัก
- **250A-630A**: โรงงาน
- **630A-1600A**: อาคารขนาดใหญ่

### ความแตกต่าง MCB vs MCCB

| หัวข้อ | MCB | MCCB |
|---|---|---|
| ขนาด | 6-63A | 50-1600A |
| Breaking Capacity | 6-10 kA | 25-100 kA |
| ปรับค่าได้ | ไม่ได้ | ได้ |
| ราคา | ถูก | แพง |
| ใช้ที่ | บ้าน | โรงงาน |

### การปรับค่า MCCB
```
┌─────────────────────────┐
│  Thermal Setting        │ ← ปรับ Overload (0.8-1.0 × In)
│  Magnetic Setting       │ ← ปรับ Short Circuit (5-10 × In)
│  Time Delay             │ ← ปรับเวลาตัด
└─────────────────────────┘
```

### ตัวอย่างการเลือก MCCB

#### ตัวอย่าง: เมนโรงงาน
```
โหลดรวม: 200A
Short Circuit: 30 kA

เลือก MCCB: 250A, 35 kA
Thermal Setting: 200A (0.8 × 250)
Magnetic Setting: 2000A (10 × 200)
```

## 3. RCCB (Residual Current Circuit Breaker)

### คำนิยาม
**RCCB** หรือ **RCD** คือเบรกเกอร์ป้องกันไฟรั่ว ตรวจจับกระแสรั่วไหลลงดิน

### สัญลักษณ์
```
  ─┤├─
   ⏚
  RCCB
```

### หลักการทำงาน
```
┌─────────────────────────┐
│   L (Live)    →  I_L    │
│                         │
│   Core Transformer      │ ← ตรวจจับ I_L - I_N
│                         │
│   N (Neutral) ←  I_N    │
└─────────────────────────┘

ถ้า I_L = I_N → ปกติ
ถ้า I_L ≠ I_N → มีกระแสรั่ว → ตัด!
```

### ความไวในการตัด (Sensitivity)

#### 10 mA (0.01A)
- ใช้กับ: ห้องน้ำ, สระว่ายน้ำ
- ความปลอดภัยสูงสุด

#### 30 mA (0.03A)
- ใช้กับ: บ้านทั่วไป
- **มาตรฐาน IEC**
- ป้องกันไฟดูด

#### 100 mA (0.1A)
- ใช้กับ: โรงงาน
- ป้องกันไฟไหม้

#### 300 mA (0.3A)
- ใช้กับ: เมนหลัก
- ป้องกันไฟไหม้

### ประเภท RCCB

#### Type AC
- ตรวจจับ: AC Residual Current
- ใช้กับ: โหลดทั่วไป
- ราคาถูก

#### Type A
- ตรวจจับ: AC + Pulsating DC
- ใช้กับ: เครื่องซักผ้า, เตาไฟฟ้า
- แนะนำสำหรับบ้าน

#### Type B
- ตรวจจับ: AC + DC
- ใช้กับ: Solar Inverter, EV Charger
- ราคาแพง

### ตัวอย่างการเลือก RCCB

#### ตัวอย่าง 1: บ้านทั่วไป
```
เมน: 50A
เลือก RCCB: 63A, 30mA, Type A
```

#### ตัวอย่าง 2: ห้องน้ำ
```
โหลด: 20A
เลือก RCCB: 25A, 10mA, Type A
```

### การทดสอบ RCCB
```
กดปุ่ม TEST ทุกเดือน
→ RCCB ต้องตัดทันที
→ ถ้าไม่ตัด = เสีย ต้องเปลี่ยน!
```

## 4. RCBO (Residual Current Breaker with Overload)

### คำนิยาม
**RCBO** = **MCB + RCCB** รวมกัน ป้องกันทั้งโอเวอร์โหลด, ลัดวงจร, และไฟรั่ว

### สัญลักษณ์
```
  ─┤├─
   ⏚
  RCBO
```

### ข้อดี
- ประหยัดพื้นที่ (1 ตัวแทน 2 ตัว)
- ป้องกันครบ
- เหมาะกับวงจรสำคัญ

### ข้อเสีย
- ราคาแพง (แพงกว่า MCB + RCCB)
- ซ่อมยาก

### ตัวอย่างการใช้ RCBO

#### ตัวอย่าง: เครื่องทำน้ำอุ่น
```
กำลัง: 3,500W
กระแส: 15.9A

เลือก RCBO: 20A, 30mA, Type C
→ ป้องกันครบทุกอย่าง
```

## 5. ELCB (Earth Leakage Circuit Breaker)

### คำนิยาม
**ELCB** คืออุปกรณ์ป้องกันไฟรั่วแบบเก่า **ไม่แนะนำให้ใช้แล้ว**

### ประเภท

#### Voltage-Operated ELCB
- ตรวจจับแรงดันที่ลงดิน
- ไม่แม่นยำ
- **เลิกใช้แล้ว**

#### Current-Operated ELCB
- เหมือน RCCB
- เปลี่ยนชื่อเป็น RCCB แล้ว

## การต่อวงจรป้องกัน

### แบบที่ 1: MCB + RCCB (แนะนำ)
```
Main RCCB (63A, 30mA)
  ├─ MCB 20A (แอร์)
  ├─ MCB 20A (เครื่องทำน้ำอุ่น)
  ├─ MCB 16A (ปลั๊ก)
  └─ MCB 10A (หลอดไฟ)
```

### แบบที่ 2: RCBO แยกวงจร
```
Main MCB (50A)
  ├─ RCBO 20A, 30mA (เครื่องทำน้ำอุ่น)
  ├─ RCBO 20A, 30mA (แอร์)
  ├─ MCB 16A (ปลั๊ก)
  └─ MCB 10A (หลอดไฟ)
```

### แบบที่ 3: โรงงาน
```
Main MCCB (250A)
  ├─ RCCB (100A, 100mA)
  │   ├─ MCB 32A (เครื่องจักร 1)
  │   └─ MCB 32A (เครื่องจักร 2)
  └─ MCCB (100A)
      ├─ MCB 20A (แสงสว่าง)
      └─ MCB 32A (ปลั๊ก)
```

## ตารางเปรียบเทียบ

| อุปกรณ์ | Overload | Short Circuit | ไฟรั่ว | ขนาด | ราคา |
|---|---|---|---|---|---|
| MCB | ✓ | ✓ | ✗ | 6-63A | ถูก |
| MCCB | ✓ | ✓ | ✗ | 50-1600A | แพง |
| RCCB | ✗ | ✗ | ✓ | 25-100A | ปานกลาง |
| RCBO | ✓ | ✓ | ✓ | 6-63A | แพง |
| ELCB | ✗ | ✗ | ✓ | - | เลิกใช้ |

## ข้อผิดพลาดที่พบบ่อย

### ❌ เลือกขนาดเล็กเกินไป
```
ผิด: แอร์ 15A ใช้ MCB 10A
ถูก: ใช้ MCB 20A (1.25 × 15 = 18.75A)
```

### ❌ ไม่ใช้ RCCB
```
ผิด: ใช้แค่ MCB
ถูก: ต้องมี RCCB ป้องกันไฟรั่ว
     (บังคับตามกฎหมาย!)
```

### ❌ ใช้ Type ผิด
```
ผิด: ใช้ Type B กับแอร์
ถูก: ใช้ Type C (แอร์มีกระแสสตาร์ทสูง)
```

### ❌ ไม่ทดสอบ RCCB
```
ผิด: ติดตั้งแล้วไม่เคยทดสอบ
ถูก: กดปุ่ม TEST ทุกเดือน
```

## Tips การเลือกใช้

### 1. บ้านทั่วไป
- เมน: RCCB 63A, 30mA, Type A
- แอร์: MCB 20A, Type C
- ปลั๊ก: MCB 16A, Type C
- หลอดไฟ: MCB 10A, Type B หรือ C

### 2. ห้องน้ำ
- RCBO 20A, 10mA, Type A
- หรือ RCCB 25A, 10mA แยกวงจร

### 3. โรงงาน
- เมน: MCCB 250A, 50 kA
- Sub-main: RCCB 100A, 100mA
- เครื่องจักร: MCB 32A, Type D
