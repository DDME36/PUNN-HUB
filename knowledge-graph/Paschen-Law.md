# Paschen's Law - กฎของพาสเชน

## คำนิยาม
**กฎของพาสเชน (Paschen's Law)** อธิบายความสัมพันธ์ระหว่างแรงดันเบรกดาวน์ของก๊าซกับผลคูณของความดันก๊าซและระยะห่างของช่องว่าง

## สูตร
$$V_b = f(pd)$$

- Vb = แรงดันเบรกดาวน์ (V)
- p = ความดันก๊าซ (Pa หรือ Torr)
- d = ระยะห่างระหว่างอิเล็กโตรด (m)
- pd = ผลคูณพาสเชน

## กราฟพาสเชน
```
แกน X: pd (ความดัน × ระยะห่าง)
แกน Y: Vb (แรงดันเบรกดาวน์)

รูปร่าง: เป็นรูป U
- มีจุดต่ำสุด (Minimum breakdown voltage)
- ซ้าย: pd ต่ำ → Vb สูง
- ขวา: pd สูง → Vb สูง
```

## ความหมาย

### pd ต่ำ (ความดันต่ำหรือระยะห่างน้อย)
- โมเลกุลก๊าซน้อย
- อิเล็กตรอนชนโมเลกุลได้น้อย
- ต้องใช้แรงดันสูงเพื่อเบรกดาวน์

### pd สูง (ความดันสูงหรือระยะห่างมาก)
- โมเลกุลก๊าซหนาแน่น
- อิเล็กตรอนชนบ่อยแต่พลังงานไม่พอ
- ต้องใช้แรงดันสูงเพื่อเบรกดาวน์

### pd ที่จุดต่ำสุด
- สภาวะที่เบรกดาวน์ง่ายที่สุด
- สำหรับอากาศ: pd ≈ 0.5-1.0 Torr·cm
- Vb(min) ≈ 327V

## การประยุกต์ใช้

### 1. การออกแบบฉนวน
```
ถ้าทราบ p และ d
→ หา Vb จากกราฟพาสเชน
→ ออกแบบให้ V < Vb
```

### 2. Spark Gap
- ออกแบบ Gap ให้เบรกดาวน์ที่แรงดันที่ต้องการ
- ปรับ d หรือ p

### 3. Vacuum Insulation
- ที่ความดันต่ำมาก (Vacuum)
- pd → 0
- Vb สูงมาก (ฉนวนดี)

## ข้อจำกัด

### ใช้ได้กับ
- ก๊าซเดี่ยว (อากาศ, N₂, SF₆)
- สนามไฟฟ้าสม่ำเสมอ
- อิเล็กโตรดเรียบ

### ใช้ไม่ได้กับ
- สนามไฟฟ้าไม่สม่ำเสมอ
- อิเล็กโตรดแหลม
- ความดันสูงมาก (> 1 atm)

## ตัวอย่าง

### ตัวอย่าง 1: อากาศที่ความดันปกติ
```
p = 760 Torr, d = 1 cm
pd = 760 Torr·cm

จากกราฟ: Vb ≈ 30 kV/cm
Vb = 30 kV
```

### ตัวอย่าง 2: Vacuum
```
p = 0.01 Torr, d = 1 cm
pd = 0.01 Torr·cm (ต่ำมาก)

Vb สูงมาก (> 100 kV)
```
