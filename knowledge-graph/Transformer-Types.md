# Transformer Types - ประเภทของหม้อแปลงไฟฟ้า

## ภาพรวม

```
┌─────────────────────────────────────────────────┐
│         ประเภทของหม้อแปลง                       │
├─────────────────────────────────────────────────┤
│                                                 │
│  1. ตามจำนวนเฟส: 1φ, 3φ                        │
│  2. ตามหน้าที่: Step-up, Step-down, Isolation  │
│  3. ตามแกน: Core, Shell                        │
```

## 1. ตามจำนวนเฟส

### Single-Phase Transformer (1 เฟส)
```
Primary  ||  Secondary
  )))    ||    (((
  N₁     ||    N₂
```

**ใช้กับ:**
- บ้านเรือน (220V)
- อุปกรณ์ขนาดเล็ก
- หม้อแปลงเสา (Pole Transformer)

**ขนาด:** 5-50 kVA

### Three-Phase Transformer (3 เฟส)
```
   R )))  ||  ((( R
   S )))  ||  ((( S
   T )))  ||  ((( T
```

**ใช้กับ:**
- โรงงาน
- อาคารขนาดใหญ่
- สถานีไฟฟ้า

**ขนาด:** 50 kVA - 500 MVA

## 2. ตามหน้าที่

### Step-Up Transformer (เพิ่มแรงดัน)
```
V₁ = 22 kV  →  V₂ = 115 kV
N₁ < N₂

ใช้ที่: โรงไฟฟ้า → สายส่ง
```

### Step-Down Transformer (ลดแรงดัน)
```
V₁ = 115 kV  →  V₂ = 22 kV
N₁ > N₂

ใช้ที่: สายส่ง → โรงงาน
```

### Isolation Transformer (แยกวงจร)
```
V₁ = 220V  →  V₂ = 220V
N₁ = N₂

ใช้ที่: 
- ป้องกันไฟรั่ว
- ลด Noise
- ห้องผ่าตัด
```

### Auto Transformer (ออโต้ทรานส์ฟอร์เมอร์)
```
    ┌───┐
V₁  │   │
    │   ├─── V₂
    │   │
    └───┘
    
ขดลวดเดียว, มี Tap
ประหยัดทองแดง 30-50%
```

**ข้อดี:**
- ราคาถูก
- ขนาดเล็ก
- ประสิทธิภาพสูง

**ข้อเสีย:**
- ไม่แยกวงจร
- อันตรายถ้าขาด Neutral

**ใช้กับ:**
- สตาร์ทมอเตอร์
- ปรับแรงดัน (Variac)

## 3. ตามโครงสร้างแกน

### Core Type (แบบแกนกลาง)
```
    ┌───┐
    │ ))) │
    │ ((( │
    └───┘
    
ขดลวดอยู่รอบแกน
ใช้กันมากที่สุด
```

### Shell Type (แบบแกนห่อหุ้ม)
```
  ┌─────┐
  │ ))) │
  │ ((( │
  └─────┘
  
แกนห่อหุ้มขดลวด
ทนแรงกลสูง
```

## 4. ตามการต่อขดลวด (3 เฟส)

### Y-Y (Star-Star)
```
Primary: Y    Secondary: Y
Neutral: มี   Neutral: มี
```

**ข้อดี:**
- มี Neutral ใช้งานได้
- ฉนวนน้อย (แรงดันต่ำ)

**ข้อเสีย:**
- มี Harmonics ลำดับที่ 3
- ต้องมี Neutral Ground

**ใช้กับ:** ระบบจำหน่าย (22kV/380V)

### Δ-Δ (Delta-Delta)
```
Primary: Δ    Secondary: Δ
Neutral: ไม่มี   Neutral: ไม่มี
```

**ข้อดี:**
- ไม่มี Harmonics ลำดับที่ 3
- ทำงานได้ถึง 58% ถ้าขดลวด 1 ตัวเสีย (Open Delta)

**ข้อเสีย:**
- ไม่มี Neutral
- กระแสในขดลวดสูง

**ใช้กับ:** โรงงาน (ไม่ต้องการ Neutral)

### Y-Δ (Star-Delta)
```
Primary: Y    Secondary: Δ
Neutral: มี   Neutral: ไม่มี
```

**ข้อดี:**
- ลด Harmonics
- มี Neutral ฝั่งปฐมภูมิ

**ใช้กับ:** Step-down (115kV/22kV)

### Δ-Y (Delta-Star)
```
Primary: Δ    Secondary: Y
Neutral: ไม่มี   Neutral: มี
```

**ข้อดี:**
- มี Neutral ฝั่งทุติยภูมิ
- ลด Harmonics

**ใช้กับ:** Step-up (22kV/115kV)

## 5. ตามการใช้งาน

### Power Transformer (หม้อแปลงกำลัง)
- **ขนาด:** > 500 kVA
- **แรงดัน:** > 33 kV
- **ใช้ที่:** สถานีไฟฟ้า, โรงไฟฟ้า
- **โหลด:** 100% ตลอดเวลา

### Distribution Transformer (หม้อแปลงจำหน่าย)
- **ขนาด:** 25-500 kVA
- **แรงดัน:** 22kV/380V
- **ใช้ที่:** เสาไฟฟ้า, ชุมชน
- **โหลด:** เปลี่ยนแปลงตามเวลา

### Instrument Transformer (หม้อแปลงวัด)
- **CT:** Current Transformer ([[CT-VT]])
- **VT/PT:** Voltage Transformer
- **ใช้ที่:** วัดกระแส/แรงดันสูง

### Furnace Transformer (หม้อแปลงเตาหลอม)
- **ขนาด:** 1-100 MVA
- **แรงดัน:** 33kV/100-1000V
- **กระแส:** สูงมาก (10,000-100,000A)
- **ใช้ที่:** โรงงานเหล็ก

### Welding Transformer (หม้อแปลงเชื่อม)
- **แรงดัน:** 220V/20-80V
- **กระแส:** 50-500A
- **ใช้ที่:** งานเชื่อม

### Rectifier Transformer (หม้อแปลงเรียงกระแส)
- **ใช้กับ:** Rectifier (AC → DC)
- **ใช้ที่:** ระบบ DC, Electroplating

## 6. ตามการระบายความร้อน

### ONAN (Oil Natural, Air Natural)
```
น้ำมันหมุนเวียนเอง
อากาศเย็นเอง
ไม่มีพัดลม, ไม่มีปั๊ม
```
**ขนาด:** < 10 MVA

### ONAF (Oil Natural, Air Forced)
```
น้ำมันหมุนเวียนเอง
อากาศเป่าด้วยพัดลม
มีพัดลม
```
**ขนาด:** 10-60 MVA

### OFAF (Oil Forced, Air Forced)
```
น้ำมันสูบด้วยปั๊ม
อากาศเป่าด้วยพัดลม
มีปั๊ม + พัดลม
```
**ขนาด:** > 60 MVA

### OFWF (Oil Forced, Water Forced)
```
น้ำมันสูบด้วยปั๊ม
น้ำเย็นสูบด้วยปั๊ม
ระบายความร้อนดีที่สุด
```
**ขนาด:** > 100 MVA

### Dry Type (แบบแห้ง)
```
ไม่ใช้น้ำมัน
ใช้อากาศระบายความร้อน
ปลอดภัย, ไม่ไฟไหม้
```
**ใช้ที่:** อาคาร, โรงพยาบาล

## 7. ตามฉนวน

### Oil-Immersed (จุ่มน้ำมัน)
- **ฉนวน:** น้ำมันหม้อแปลง
- **ข้อดี:** ระบายความร้อนดี, ราคาถูก
- **ข้อเสีย:** ไฟไหม้ได้, ต้องบำรุงรักษา

### Dry Type (แบบแห้ง)
- **ฉนวน:** อากาศ, Resin
- **ข้อดี:** ปลอดภัย, ไม่ไฟไหม้
- **ข้อเสีย:** ราคาแพง, ระบายความร้อนแย่

### Gas-Insulated (ฉนวนก๊าซ)
- **ฉนวน:** SF₆ Gas
- **ข้อดี:** ขนาดเล็ก, ทนแรงดันสูง
- **ข้อเสีย:** ราคาแพงมาก

## 8. หม้อแปลงพิเศษ

### Scott-T Transformer
```
แปลง 3 เฟส → 2 เฟส
ใช้กับ: รถไฟฟ้า
```

### Zig-Zag Transformer
```
สร้าง Neutral Point
ลด Harmonics
ใช้กับ: ระบบไม่มี Neutral
```

### Grounding Transformer
```
สร้างจุดลงดิน
ใช้กับ: ระบบ Delta
```

### Phase-Shifting Transformer
```
เลื่อนมุมเฟส
ควบคุมการไหลของกำลังไฟฟ้า
ใช้กับ: สายส่งขนาดใหญ่
```

## ตารางเปรียบเทียบการต่อ

| การต่อ | Neutral | Harmonics | ใช้กับ |
|---|---|---|---|
| Y-Y | มี | มี | ระบบจำหน่าย |
| Δ-Δ | ไม่มี | ไม่มี | โรงงาน |
| Y-Δ | ปฐมภูมิ | ลด | Step-down |
| Δ-Y | ทุติยภูมิ | ลด | Step-up |

## ตารางเปรียบเทียบการระบายความร้อน

| ชนิด | ปั๊ม | พัดลม | ขนาด | ราคา |
|---|---|---|---|---|
| ONAN | ✗ | ✗ | < 10 MVA | ถูก |
| ONAF | ✗ | ✓ | 10-60 MVA | ปานกลาง |
| OFAF | ✓ | ✓ | > 60 MVA | แพง |
| OFWF | ✓ | ✓ | > 100 MVA | แพงมาก |

## ข้อผิดพลาดที่พบบ่อย

### ❌ เลือกการต่อผิด
```
ผิด: ใช้ Y-Y โดยไม่มี Neutral Ground
ถูก: Y-Y ต้องมี Neutral Ground
     หรือใช้ Δ-Y, Y-Δ แทน
```

### ❌ ใช้ Auto Transformer ผิด
```
ผิด: ใช้ Auto Transformer แยกวงจร
ถูก: Auto Transformer ไม่แยกวงจร
     ใช้ Isolation Transformer แทน
```

### ❌ ไม่ดูการระบายความร้อน
```
ผิด: ใช้ ONAN กับโหลด 100%
ถูก: ONAN ใช้กับโหลดเปลี่ยนแปลง
     โหลด 100% ใช้ ONAF ขึ้นไป
```

## Tips การเลือกหม้อแปลง

- 22kV/220V
- Pole Transformer

- 3 เฟส, 100-500 kVA
- 22kV/380V
- Dry Type (ปลอดภัย)

### 3. โรงงาน
- 3 เฟส, 500-2000 kVA
- 22kV/380V
- Oil-Immersed, ONAF

### 4. สถานีไฟฟ้า
- 3 เฟส, > 10 MVA
- 115kV/22kV
- Oil-Immersed, OFAF
