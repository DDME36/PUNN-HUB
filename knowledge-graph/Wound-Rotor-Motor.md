# Wound Rotor Motor - มอเตอร์โรเตอร์พันขดลวด

## คำนิยาม
**Wound Rotor Motor** คือ [[Motor]] เหนี่ยวนำที่โรเตอร์พันด้วยขดลวด เชื่อมต่อออกมาผ่าน Slip Ring สามารถควบคุมได้

## โครงสร้าง

### Stator
- เหมือน [[Squirrel-Cage-Motor]]
- ขดลวด 3 เฟส

### Rotor
- พันขดลวด 3 เฟส
- เชื่อมต่อกับ Slip Ring
- ต่อกับ External Resistor

### Slip Ring & Brush
- 3 วง (3 เฟส)
- แปรงถ่านสัมผัส
- ดึงกระแสออกมา

## หลักการทำงาน

### การควบคุม
```
เพิ่ม External Resistor
→ เพิ่มความต้านทานโรเตอร์
→ เพิ่ม Slip
→ ลดความเร็ว
→ เพิ่มแรงบิดสตาร์ท
```

## ข้อดี

### 1. แรงบิดสตาร์ทสูง
- ปรับ Resistor ให้สูง
- แรงบิดสตาร์ทสูงมาก
- เหมาะกับโหลดหนัก

### 2. กระแสสตาร์ทต่ำ
- Resistor จำกัดกระแส
- ต่ำกว่า [[Squirrel-Cage-Motor]]

### 3. ควบคุมความเร็วได้
- ปรับ Resistor
- ควบคุมความเร็วได้
- แต่สูญเสียพลังงาน

### 4. สตาร์ทนุ่มนวล
- ปรับ Resistor ค่อยๆ
- สตาร์ทนุ่ม

## ข้อเสีย

### 1. ราคาแพง
- โครงสร้างซับซ้อน
- มี Slip Ring
- แพงกว่า Squirrel Cage มาก

### 2. บำรุงรักษายาก
- มี Slip Ring และ Brush
- ต้องเปลี่ยนแปรง
- ต้องทำความสะอาด

### 3. ขนาดใหญ่
- ใหญ่กว่า Squirrel Cage
- หนักกว่า

### 4. สูญเสียพลังงาน
- ถ้าใช้ควบคุมความเร็ว
- สูญเสียใน Resistor
- ประสิทธิภาพต่ำ

## การใช้งาน

### เหมาะกับ
- **เครน, รถยก**: ต้องการแรงบิดสูง
- **ลิฟต์**: สตาร์ท-หยุดบ่อย
- **โรงสี**: โหลดหนัก
- **เครื่องจักรขนาดใหญ่**

### ไม่เหมาะกับ
- งานทั่วไป (ใช้ Squirrel Cage)
- งานที่ต้องการบำรุงรักษาน้อย
- งบประมาณจำกัด

## การสตาร์ท

### ขั้นตอน
```
1. Resistor สูงสุด
2. สตาร์ทมอเตอร์
3. ลด Resistor ค่อยๆ
4. Short Circuit Resistor
5. ทำงานปกติ
```

## ปัจจุบัน
- ใช้น้อยลงมาก
- ถูกแทนที่ด้วย [[VSD]]
- VSD ควบคุมได้ดีกว่า
- ไม่ต้องบำรุงรักษา
