# Delta Connection (Δ) - การต่อแบบเดลต้า

## คำนิยาม
**การต่อแบบเดลต้า (Delta Connection)** คือการต่อขดลวด 3 เฟสเป็นรูปสามเหลี่ยม โดยต่อปลายของเฟสหนึ่งเข้ากับต้นของเฟสถัดไป

## สูตรหลัก

### แรงดัน
$$V_{line} = V_{phase}$$
$$V_L = V_P$$

### กระแส
$$I_{line} = \sqrt{3} \times I_{phase}$$
$$I_L = 1.732 \times I_P$$

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ ระบบ 3 เฟส

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หากระแสสาย
```
I_phase = 100A (กระแสในขดลวด)
I_line = √3 × I_phase
I_line = 1.732 × 100 = 173.2A
```

### ตัวอย่าง 2: หม้อแปลง
```
S = 100 kVA, V_L = 380V (Delta)
I_L = S / (√3 × V_L)
I_L = 100,000 / (1.732 × 380) = 152A

I_phase = I_L / √3
I_phase = 152 / 1.732 = 87.8A
```

### ตัวอย่าง 3: หากำลัง
```
V_L = 380V, I_L = 100A, PF = 0.85
P = √3 × V_L × I_L × cosθ
P = 1.732 × 380 × 100 × 0.85
P = 55,944 W ≈ 56 kW
```

## ข้อดี
- ไม่มี Neutral → ประหยัดสาย
- กระแสเฟสต่ำกว่า → ขดลวดเล็กกว่า
- เหมาะกับโหลดสมดุล

## ข้อเสีย
- ไม่มี Neutral → ดึงไฟ 1 เฟสไม่ได้
- แรงดันเฟสสูง → ฉนวนต้องหนากว่า

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** I_L = I_P (ใน Delta)  
✅ **ถูก:** I_L = √3 × I_P

❌ **ผิด:** V_L = √3 × V_P (ใน Delta)  
✅ **ถูก:** V_L = V_P (เท่ากัน!)
