# Capacitor Bank - ตู้คาปาซิเตอร์

## คำนิยาม
**ตู้คาปาซิเตอร์ (Capacitor Bank)** คือชุดของ [[Capacitor-C]] หลายตัวที่ต่อรวมกัน ใช้สำหรับ [[Power-Factor-Correction]] ในโรงงานและระบบไฟฟ้า

## วัตถุประสงค์
1. **แก้ Power Factor ต่ำ**
2. **ลดกระแสสาย**
3. **ลดค่าไฟ** (หลีกเลี่ยงค่าปรับ PF)
4. **ลดการสูญเสียในสาย**

## หลักการทำงาน
- [[Motor]] และ [[Transformer]] สร้าง [[Reactive-Power-Q]] ล้าหลัง (+jQ)
- [[Capacitor-Bank]] สร้าง [[Reactive-Power-Q]] นำหน้า (-jQ)
- +jQ และ -jQ หักล้างกัน
- [[Power-Factor-PF]] เพิ่มขึ้น

## สูตรการคำนวณ

### หาขนาด Capacitor Bank
$$Q_C = P(\tan\theta_1 - \tan\theta_2)$$

$$Q_C = P\left(\tan(\cos^{-1}PF_1) - \tan(\cos^{-1}PF_2)\right)$$

### หาความจุ (Capacitance)
$$C = \frac{Q_C}{2\pi f V^2}$$
- $Q_C$ = กำลังรีแอคทีฟ (VAR)
- f = ความถี่ (Hz)
- V = แรงดัน (V)

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ พื้นฐาน
- [[Capacitor-Bank]] → ประกอบด้วย [[Capacitor-C]] หลายตัว
- [[Capacitor-Bank]] → ใช้แก้ [[Power-Factor-PF]] ต่ำ
- [[Capacitor-Bank]] → ลด [[Reactive-Power-Q]] ล้าหลัง

### 🔺 สามเหลี่ยมกำลัง
```
ก่อนติดตั้ง:
S₁ = √(P² + Q₁²), PF₁ = P/S₁

หลังติดตั้ง:
Q₂ = Q₁ - Q_C
S₂ = √(P² + Q₂²), PF₂ = P/S₂
```

### ⚙️ การใช้งาน
- แก้ PF ของ [[Motor]]
- แก้ PF ของ [[Transformer]]
- ลดกระแสใน [[Three-Phase-System]]

## ประเภทของ Capacitor Bank

### 1. Fixed Capacitor Bank
- ขนาดคงที่
- เปิด-ปิดด้วย Contactor
- ราคาถูก

### 2. Automatic Capacitor Bank
- ปรับขนาดอัตโนมัติ
- มี PF Controller
- ราคาแพงกว่า

### 3. Detuned Capacitor Bank
- มี Reactor ต่อแบบอนุกรม
- ป้องกัน Harmonic
- ใช้กับโหลด Non-linear

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หาขนาด Capacitor Bank
```
โรงงานใช้ไฟ: P = 100 kW
PF เดิม = 0.7 (ล้าหลัง)
ต้องการ PF = 0.95

Q_C = P(tan θ₁ - tan θ₂)
θ₁ = cos⁻¹(0.7) = 45.57°
θ₂ = cos⁻¹(0.95) = 18.19°

Q_C = 100(tan 45.57° - tan 18.19°)
Q_C = 100(1.020 - 0.329)
Q_C = 69.1 kVAR

ติดตั้ง Capacitor Bank 70 kVAR
```

### ตัวอย่าง 2: หาความจุ
```
Q_C = 70 kVAR, V = 380V, f = 50Hz

C = Q_C / (2πfV²)
C = 70,000 / (2π × 50 × 380²)
C = 70,000 / 45,553,093
C = 0.001536 F = 1,536 μF
```

### ตัวอย่าง 3: ผลลัพธ์หลังติดตั้ง
```
ก่อน:
P = 100 kW, PF = 0.7
S₁ = P/PF = 100/0.7 = 142.9 kVA
I₁ = S₁/(√3 × V) = 142,900/(1.732 × 380) = 217A

หลัง:
P = 100 kW, PF = 0.95
S₂ = P/PF = 100/0.95 = 105.3 kVA
I₂ = S₂/(√3 × V) = 105,300/(1.732 × 380) = 160A

ลดกระแสได้: 217 - 160 = 57A (26%)
```

## ประโยชน์ที่ได้รับ

### 1. ลดค่าไฟ
```
ค่าปรับ PF (ถ้า PF < 0.85):
ค่าปรับ = ค่าไฟ × (0.85/PF - 1)

ตัวอย่าง:
ค่าไฟ = 50,000 บาท, PF = 0.7
ค่าปรับ = 50,000 × (0.85/0.7 - 1)
ค่าปรับ = 50,000 × 0.214 = 10,700 บาท

หลังแก้ PF = 0.95 → ไม่มีค่าปรับ
ประหยัด 10,700 บาท/เดือน
```

### 2. ลดการสูญเสีย
```
P_loss = I²R
ลดกระแส 26% → ลดสูญเสีย (0.74)² = 55%
```

### 3. เพิ่มกำลังไฟฟ้าที่ใช้ได้
```
หม้อแปลง 500 kVA
PF = 0.7 → ใช้ได้ 350 kW
PF = 0.95 → ใช้ได้ 475 kW
เพิ่ม 125 kW!
```

## การติดตั้ง

### ตำแหน่งติดตั้ง
1. **ที่ Main Distribution Board**: แก้ PF รวมทั้งโรงงาน
2. **ที่มอเตอร์แต่ละตัว**: แก้ PF เฉพาะจุด

### ขนาดที่เหมาะสม
- **ไม่ควรแก้เกิน PF = 0.95-0.98**
- ถ้าแก้เกิน → PF นำหน้า → ปัญหาใหม่!

## ข้อควรระวัง

### ⚠️ Over-Compensation
```
ผิด: ติดตั้ง Capacitor ใหญ่เกินไป
ผล: PF นำหน้า (Leading) → ปัญหาแรงดันสูง
ถูก: แก้ PF ให้อยู่ที่ 0.95-0.98
```

### ⚠️ Harmonic
```
ถ้ามี VSD, UPS, Rectifier (โหลด Non-linear)
→ ใช้ Detuned Capacitor Bank
→ มี Reactor ป้องกัน Resonance
```

### ⚠️ Switching
```
เปิด-ปิด Capacitor Bank บ่อย
→ เกิด Inrush Current
→ ใช้ Contactor ที่รองรับ Capacitor
```

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** ติดตั้ง Capacitor แล้ว PF = 1.0  
✅ **ถูก:** แก้ PF ให้อยู่ที่ 0.95-0.98 (ไม่ควรเกิน)

❌ **ผิด:** คิดว่า Capacitor ช่วยประหยัดไฟ  
✅ **ถูก:** Capacitor ลดกระแส → ลดค่าปรับ + ลดสูญเสีย

❌ **ผิด:** ใช้ Capacitor ขนาดเดียวกับ kW ของมอเตอร์  
✅ **ถูก:** ต้องคำนวณจาก PF เดิมและ PF เป้าหมาย
