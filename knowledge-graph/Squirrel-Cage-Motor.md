# Squirrel-Cage Motor - มอเตอร์กรงกระรอก

## คำนิยาม
**มอเตอร์กรงกระรอก (Squirrel-Cage Motor)** คือ [[Motor]] เหนี่ยวนำชนิดที่โรเตอร์ทำจากแท่งทองแดงหรืออลูมิเนียมเชื่อมต่อกันเป็นรูปกรงกระรอก

## โครงสร้างโรเตอร์

### รูปร่าง
```
แท่งโลหะ (Rotor Bars)
  |  |  |  |  |  |
  |  |  |  |  |  |
====================  End Ring (บน)
  |  |  |  |  |  |
  |  |  |  |  |  |
====================  End Ring (ล่าง)

คล้ายกรงกระรอก → ชื่อ Squirrel Cage
```

### วัสดุ
- **อลูมิเนียม**: ราคาถูก, ใช้ทั่วไป
- **ทองแดง**: ประสิทธิภาพสูง, ราคาแพง
- **Die-cast**: หล่อเป็นชิ้นเดียว

## หลักการทำงาน

### การเหนี่ยวนำ
1. สนามแม่เหล็กหมุนจาก Stator
2. ตัดผ่านแท่งโรเตอร์
3. เกิดกระแสเหนี่ยวนำในแท่ง
4. กระแส × สนามแม่เหล็ก → แรงบิด
5. โรเตอร์หมุนตามสนามแม่เหล็ก

### Slip
- [[Slip]] = 3-5% (โหลดเต็ม)
- ไม่เคยหมุนเท่า [[Synchronous-Speed]]
- ถ้า Slip = 0 → ไม่มีกระแสเหนี่ยวนำ → ไม่มีแรงบิด

## ข้อดี

### 1. ทนทาน
- ไม่มี [[Commutator]]
- ไม่มีแปรงถ่าน
- ไม่มีส่วนสัมผัส
- อายุการใช้งานยาว

### 2. บำรุงรักษาง่าย
- ไม่ต้องเปลี่ยนแปรง
- ไม่ต้องทำความสะอาด Commutator
- เปลี่ยนแบริ่งเป็นครั้งคราว

### 3. ราคาถูก
- โครงสร้างง่าย
- ผลิตง่าย
- ราคาถูกที่สุด

### 4. เชื่อถือได้
- ไม่มีประกายไฟ
- ใช้ในสภาพแวดล้อมที่มีฝุ่น/ก๊าซได้
- ทนทานต่อโอเวอร์โหลด

## ข้อเสีย

### 1. กระแสสตาร์ทสูง
- 5-7 เท่าของกระแสทำงาน
- ทำให้แรงดันตก
- ต้องใช้ [[Star-Delta-Starter]] หรือ [[Soft-Starter]]

### 2. แรงบิดสตาร์ทต่ำ
- ประมาณ 1.5-2 เท่าของแรงบิดเต็มโหลด
- ไม่เหมาะกับโหลดหนัก
- เทียบกับ [[DC-Motor]] แรงบิดต่ำกว่า

### 3. ควบคุมความเร็วยาก
- ความเร็วขึ้นกับความถี่
- ต้องใช้ [[VSD]] ถ้าต้องการปรับความเร็ว
- ไม่เหมาะกับงานที่ต้องปรับความเร็วบ่อย

## การสตาร์ท

### 1. DOL (Direct Online)
- ต่อตรงเข้าไฟ
- กระแสสูงมาก
- ใช้กับมอเตอร์เล็ก (< 5 HP)

### 2. Star-Delta
- สตาร์ทด้วย [[Star-Connection]]
- เปลี่ยนเป็น [[Delta-Connection]]
- ลดกระแสเหลือ 1/3

### 3. Soft Starter
- ใช้ Thyristor ควบคุมแรงดัน
- เพิ่มแรงดันค่อยๆ
- สตาร์ทนุ่มนวล

### 4. VSD
- ควบคุมความถี่
- สตาร์ทนุ่มที่สุด
- ปรับความเร็วได้

## การใช้งาน

### เหมาะกับ
- พัดลม, ปั๊มน้ำ
- คอมเพรสเซอร์
- สายพานลำเลียง
- เครื่องจักรทั่วไป

### ไม่เหมาะกับ
- รถยก, เครน (ต้องการแรงบิดสูง)
- งานที่ต้องปรับความเร็วบ่อย
- งานที่ต้องการความแม่นยำสูง
