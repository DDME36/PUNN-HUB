# Soft Starter - สตาร์ทเตอร์แบบอ่อน

## คำนิยาม
**Soft Starter** คืออุปกรณ์สตาร์ท [[Motor]] โดยเพิ่มแรงดันค่อยๆ ด้วย Thyristor/SCR ทำให้สตาร์ทนุ่มนวล

## หลักการทำงาน

### อุปกรณ์หลัก
- **Thyristor (SCR)**: สวิตช์กึ่งตัวนำ
- **Control Circuit**: ควบคุมมุมจุดชนวน
- **Bypass Contactor**: ต่อตรงเมื่อหมุนเต็มที่

### การทำงาน
```
1. เริ่มต้น: แรงดัน 30-40%
2. เพิ่มแรงดันค่อยๆ (Ramp)
3. ถึง 100% ใน 5-30 วินาที
4. Bypass Contactor ทำงาน
5. Thyristor หยุดทำงาน
```

## การควบคุม

### Voltage Ramp
- เพิ่มแรงดันเป็นเส้นตรง
- ปรับเวลา Ramp ได้

### Current Limit
- จำกัดกระแสสูงสุด
- ป้องกันโอเวอร์โหลด

### Torque Control
- ควบคุมแรงบิด
- เหมาะกับโหลดหนัก

## ข้อดี

### 1. สตาร์ทนุ่มนวล
- ไม่มีกระแสกระชาก
- ไม่มีแรงบิดกระชาก
- ลดการสั่นสะเทือน

### 2. ลดกระแสสตาร์ท
- 2-4 เท่าของกระแสทำงาน
- ดีกว่า DOL (5-7×)
- แต่สูงกว่า [[Star-Delta-Starter]] (1.7-2.3×)

### 3. ปรับได้
- ปรับเวลา Ramp
- ปรับกระแสสูงสุด
- ปรับแรงบิดเริ่มต้น

### 4. ป้องกันในตัว
- Overload
- Phase Loss
- Overtemperature

## ข้อเสีย

### 1. ราคา
- แพงกว่า [[Star-Delta-Starter]]
- ถูกกว่า [[VSD]]

### 2. ความร้อน
- Thyristor ร้อน
- ต้องระบายความร้อน

### 3. Harmonics
- สร้าง [[Harmonics]]
- รบกวนระบบ

### 4. ไม่ปรับความเร็ว
- ใช้สตาร์ทอย่างเดียว
- ไม่ควบคุมความเร็วได้

## การติดตั้ง

### ตำแหน่ง
- ใกล้มอเตอร์
- ระบายความร้อนดี
- ไม่ร้อนเกิน 40°C

### การต่อ
```
Main → Soft Starter → Motor
         |
    Bypass Contactor
```

## การตั้งค่า

### พารามิเตอร์
1. **Initial Voltage**: 30-50%
2. **Ramp Time**: 5-30 s
3. **Current Limit**: 2-4× In
4. **Stop Mode**: Soft Stop หรือ Coast

## เปรียบเทียบ

| วิธี | I_start | ราคา | ปรับความเร็ว |
|---|---|---|---|
| DOL | 5-7× | ถูกสุด | ไม่ได้ |
| Star-Delta | 1.7-2.3× | ถูก | ไม่ได้ |
| Soft Starter | 2-4× | ปานกลาง | ไม่ได้ |
| [[VSD]] | 1-1.5× | แพง | ได้ |
