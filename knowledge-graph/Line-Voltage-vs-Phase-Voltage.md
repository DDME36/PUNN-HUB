# Line Voltage vs Phase Voltage

## คำนิยาม
- **Line Voltage (VL)**: แรงดันระหว่างสาย 2 เส้น (เฟสกับเฟส)
- **Phase Voltage (VP)**: แรงดันของแต่ละเฟส (เฟสกับนิวทรัล)

## ความสัมพันธ์

### Star Connection (Y)
$$V_L = \sqrt{3} \times V_P$$
$$V_L = 1.732 \times V_P$$

ตัวอย่าง:
- VP = 220V → VL = 380V
- VP = 127V → VL = 220V

### Delta Connection (Δ)
$$V_L = V_P$$

ตัวอย่าง:
- VP = 380V → VL = 380V

## การวัด
- **VL**: วัดระหว่างเฟส A-B, B-C, C-A
- **VP**: วัดระหว่างเฟสกับนิวทรัล A-N, B-N, C-N

## ระบบไทย
```
Star Connection:
- VL = 380V (3-Phase)
- VP = 220V (1-Phase)
- 380V = 220V × √3
```
