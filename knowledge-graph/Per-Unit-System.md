# Per-Unit System - ระบบเปอร์ยูนิต

## คำนิยาม
**ระบบเปอร์ยูนิต (Per-Unit System, pu)** คือการแปลงค่าจริง (V, I, Z, S) ให้เป็นสัดส่วนเทียบกับค่าฐาน ทำให้คำนวณ [[Transformer]] หลายตัวที่ต่อกันได้ง่ายขึ้น

## สูตรพื้นฐาน
$$\text{pu} = \frac{\text{ค่าจริง (Actual Value)}}{\text{ค่าฐาน (Base Value)}}$$

## การเลือกค่าฐาน

### ค่าฐานหลัก (เลือก 2 ค่า)
1. **$S_{base}$**: กำลังฐาน (VA, kVA, MVA)
2. **$V_{base}$**: แรงดันฐาน (V, kV)

### ค่าฐานที่คำนวณได้
$$I_{base} = \frac{S_{base}}{V_{base}}$$ (1-phase)

$$I_{base} = \frac{S_{base}}{\sqrt{3} \times V_{base}}$$ (3-phase)

$$Z_{base} = \frac{V_{base}^2}{S_{base}}$$

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ การใช้งาน
- [[Per-Unit-System]] → ใช้กับ [[Transformer]] หลายตัว
- [[Per-Unit-System]] → ใช้กับ [[Percent-Impedance-Z]]
- [[Per-Unit-System]] → ใช้ในระบบ [[Three-Phase-System]]

### 🔌 ข้อดี
1. ไม่ต้องคำนวณ [[Turns-Ratio]] ข้ามหม้อแปลง
2. ค่า pu เหมือนกันทั้งสองฝั่งหม้อแปลง
3. ตรวจสอบความสมเหตุสมผลได้ง่าย
4. ใช้ในการวิเคราะห์ระบบไฟฟ้า

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: แปลงค่าเป็น pu
```
ระบบ: S_base = 100 MVA, V_base = 22 kV

Z_base = V²/S = (22,000)² / 100,000,000
Z_base = 4.84 Ω

ถ้า Z_actual = 2.42 Ω
Z_pu = Z_actual / Z_base = 2.42 / 4.84 = 0.5 pu
```

### ตัวอย่าง 2: หม้อแปลง %Z
```
หม้อแปลง 50 MVA, 115/22 kV, %Z = 10%

%Z = 10% = 0.10 pu

ถ้าเปลี่ยนฐาน:
S_base(new) = 100 MVA
V_base(new) = 115 kV (ฝั่งเดิม)

Z_pu(new) = Z_pu(old) × (V_old/V_new)² × (S_new/S_old)
Z_pu(new) = 0.10 × (115/115)² × (100/50)
Z_pu(new) = 0.10 × 1 × 2 = 0.20 pu
```

### ตัวอย่าง 3: ระบบหลายหม้อแปลง
```
G: 100 MVA, 22 kV, X = 0.15 pu
T1: 50 MVA, 22/115 kV, X = 0.10 pu
T2: 50 MVA, 115/11 kV, X = 0.10 pu

เลือกฐาน: S_base = 100 MVA
V_base = 22 kV (zone 1), 115 kV (zone 2), 11 kV (zone 3)

แปลง T1 และ T2 เป็นฐานใหม่:
X_T1(new) = 0.10 × (100/50) = 0.20 pu
X_T2(new) = 0.10 × (100/50) = 0.20 pu

X_total = X_G + X_T1 + X_T2
X_total = 0.15 + 0.20 + 0.20 = 0.55 pu
```

## สูตรแปลงฐาน (Change of Base) ⭐

### สูตรสำคัญ (ออกสอบบ่อย!)
$$Z_{pu(new)} = Z_{pu(old)} \times \left(\frac{V_{base(old)}}{V_{base(new)}}\right)^2 \times \left(\frac{S_{base(new)}}{S_{base(old)}}\right)$$

### จำง่ายๆ
- **S เพิ่ม** → Z_pu เพิ่ม (สัดส่วนตรง)
- **V เพิ่ม** → Z_pu ลด (สัดส่วนกลับกำลังสอง)

## ตัวอย่างการใช้งานจริง

### ตัวอย่าง 4: หากระแสลัดวงจร
```
ระบบ: 100 MVA, 22 kV
X_total = 0.55 pu (จากตัวอย่าง 3)

I_base = S_base / (√3 × V_base)
I_base = 100,000,000 / (1.732 × 22,000)
I_base = 2,625 A

I_fault(pu) = 1 / X_total = 1 / 0.55 = 1.82 pu
I_fault(actual) = 1.82 × 2,625 = 4,777 A
```

## ค่า pu ทั่วไป

### แรงดัน
- **ปกติ**: 0.95 - 1.05 pu
- **สูงเกิน**: > 1.1 pu (อันตราย)
- **ต่ำเกิน**: < 0.9 pu (มอเตอร์ร้อน)

### กระแส
- **ปกติ**: 0 - 1.0 pu
- **โอเวอร์โหลด**: > 1.0 pu
- **ลัดวงจร**: 10-20 pu

### อิมพีแดนซ์
- **เครื่องกำเนิด**: 0.1 - 0.3 pu
- **หม้อแปลง**: 0.05 - 0.15 pu
- **สายส่ง**: 0.01 - 0.1 pu/km

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** ลืมแปลงฐานก่อนรวม Z  
✅ **ถูก:** ต้องแปลงให้ฐานเดียวกันก่อน

❌ **ผิด:** ใช้ V_base ผิดโซน  
✅ **ถูก:** แต่ละโซนมี V_base ต่างกัน (ตาม Turns Ratio)

❌ **ผิด:** สูตรแปลงฐาน: $Z_{new} = Z_{old} \times \frac{S_{new}}{S_{old}} \times \frac{V_{new}}{V_{old}}$  
✅ **ถูก:** $Z_{new} = Z_{old} \times \frac{S_{new}}{S_{old}} \times \left(\frac{V_{old}}{V_{new}}\right)^2$

## เทคนิคจำ

### 🎯 การแปลงฐาน
```
S เพิ่ม → Z เพิ่ม (คูณ)
V เพิ่ม → Z ลด (หาร²)

ตัวอย่าง:
S: 50 → 100 MVA (×2)
V: 22 → 22 kV (×1)
Z: 0.10 → 0.10 × 2 × 1² = 0.20 pu
```

### 🎯 ตรวจสอบ
```
ค่า pu ควรอยู่ในช่วง:
- Z: 0.01 - 0.3 pu
- V: 0.95 - 1.05 pu
- I: 0 - 1.0 pu (ปกติ)
```

## ประโยชน์
1. **ง่ายต่อการคำนวณ**: ไม่ต้องคิด Turns Ratio
2. **ตรวจสอบง่าย**: ค่าผิดปกติเห็นชัด
3. **มาตรฐาน**: ใช้เปรียบเทียบอุปกรณ์ต่างขนาด
4. **ระบบใหญ่**: จำเป็นสำหรับวิเคราะห์ระบบไฟฟ้า
