# Energy Meter - มิเตอร์วัดหน่วยไฟฟ้า

## คำนิยาม
**มิเตอร์วัดหน่วยไฟฟ้า (Energy Meter)** คืออุปกรณ์วัดพลังงานไฟฟ้าที่ใช้ไป มีหน่วยเป็น **kWh (กิโลวัตต์-ชั่วโมง)**

## หน่วยวัด

### kWh (Kilowatt-hour)
$\text{Energy (kWh)} = \text{Power (kW)} \times \text{Time (h)}$

ตัวอย่าง:
```
เครื่องปรับอากาศ 2,000W ใช้ 5 ชั่วโมง
Energy = 2 kW × 5 h = 10 kWh
```

### ความสัมพันธ์
- **kWh** = พลังงาน (Energy)
- **kW** = กำลัง (Power)
- **kVA** = กำลังปรากฏ ([[Apparent-Power-S]])

## ประเภทมิเตอร์

### 1. ตามเฟส
- **1 เฟส 2 สาย**: บ้านเรือน (220V)
- **3 เฟส 4 สาย**: โรงงาน (380V)

### 2. ตามการต่อ
- **ต่อตรง**: กระแส < 100A
- **ต่อผ่าน CT-VT**: กระแสสูง ([[CT-VT]])

### 3. ตามเทคโนโลยี
- **Mechanical (แบบจานหมุน)**: เก่า
- **Electronic (แบบดิจิทัล)**: ใหม่, แม่นยำ
- **Smart Meter**: วัดได้ทั้ง 2 ทาง, ส่งข้อมูลได้

## การอ่านค่า

### มิเตอร์แบบจานหมุน
```
หมุน 1 รอบ = X kWh (ดูที่จาน)
เช่น: 1 รอบ = 0.001 kWh

นับรอบ → คำนวณหน่วย
```

### มิเตอร์ดิจิทัล
```
แสดงผลตรง: 1234.5 kWh
อ่านง่าย, แม่นยำ
```

## การคำนวณค่าไฟ

### ขั้นตอน
1. อ่านมิเตอร์เดือนนี้
2. ลบด้วยมิเตอร์เดือนที่แล้ว
3. คูณด้วยอัตราค่าไฟ (บาท/kWh)

### ตัวอย่าง
```
มิเตอร์เดือนนี้: 5,432 kWh
มิเตอร์เดือนที่แล้ว: 5,132 kWh
หน่วยที่ใช้ = 5,432 - 5,132 = 300 kWh

อัตรา 4 บาท/kWh
ค่าไฟ = 300 × 4 = 1,200 บาท
```

## การต่อมิเตอร์

### 1 เฟส 2 สาย
```
L (สายไฟ) → มิเตอร์ → โหลด
N (สายกลาง) → โหลด
```

### 3 เฟส 4 สาย
```
R, S, T (3 สายไฟ) → มิเตอร์ → โหลด
N (สายกลาง) → มิเตอร์ → โหลด
```

## การต่อผ่าน CT-VT

### สูตรคำนวณ
$\text{หน่วยจริง} = \text{หน่วยที่วัด} \times \text{CT ratio} \times \text{VT ratio}$

### ตัวอย่าง
```
มิเตอร์แสดง: 100 kWh
CT: 200/5A (Ratio = 40)
VT: 22000/110V (Ratio = 200)

หน่วยจริง = 100 × 40 × 200
หน่วยจริง = 800,000 kWh
```

### กรณีต่อตรง (ไม่มี VT)
```
มิเตอร์แสดง: 50 kWh
CT: 100/5A (Ratio = 20)
VT: ไม่มี (Ratio = 1)

หน่วยจริง = 50 × 20 × 1
หน่วยจริง = 1,000 kWh
```

## ขนาดมิเตอร์

### การเลือกขนาด
```
ขนาดมิเตอร์ = 15(45)A

15A: กระแสทำงานปกติ (Base Current)
45A: กระแสสูงสุด (Max Current)
```

### ตัวอย่างการเลือก
```
โหลดรวม = 30A
เลือกมิเตอร์ ≥ 30A
→ 30(100)A หรือ 50(100)A
```

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ พื้นฐาน
- [[Energy-Meter]] → วัด [[Electrical-Power-P]] × เวลา
- [[Energy-Meter]] → ใช้ [[CT-VT]] สำหรับกระแสสูง
- [[Energy-Meter]] → วัดเฉพาะ [[Real-Power-P]] (kW)

### 🔌 ระบบไฟฟ้า
- ใช้ใน [[Single-Phase-System]]
- ใช้ใน [[Three-Phase-System]]
- คำนวณ [[Power-Factor-PF]] ได้ (Smart Meter)

## Maximum Demand (MD)

### คำนิยาม
**MD (Maximum Demand)** = กำลังไฟฟ้าสูงสุดที่ใช้ในช่วงเวลาหนึ่ง (มักเป็น 15 นาที)

### การคิดค่า MD
```
ค่าไฟ = ค่าหน่วย (kWh) + ค่า MD (kW)

ตัวอย่าง:
หน่วย: 10,000 kWh × 4 = 40,000 บาท
MD: 50 kW × 200 = 10,000 บาท
รวม: 50,000 บาท
```

### การลด MD
- กระจายการใช้ไฟ
- ไม่เปิดเครื่องจักรพร้อมกัน
- ใช้ [[Soft-Starter]] หรือ [[VSD]]

## Time of Use (TOU)

### อัตราตามเวลา
```
On-Peak (9:00-22:00): 5 บาท/kWh
Off-Peak (22:00-9:00): 2.5 บาท/kWh

ประหยัด: ใช้ไฟช่วง Off-Peak
```

## การตรวจสอบความถูกต้อง

### วิธีตรวจ
1. ปิดโหลดทั้งหมด → มิเตอร์ต้องหยุด
2. เปิดโหลดที่รู้กำลัง → คำนวณเทียบ
3. ใช้ Clamp Meter วัดกระแส

### ตัวอย่างการตรวจ
```
เปิดหลอดไฟ 1,000W เป็นเวลา 1 ชั่วโมง
มิเตอร์ต้องเพิ่ม 1 kWh

ถ้าไม่ตรง → มิเตอร์ผิดปกติ
```

## ข้อผิดพลาดที่พบบ่อย

### ❌ สับสน kW กับ kWh
```
ผิด: เครื่องปรับอากาศ 2,000 kWh
ถูก: เครื่องปรับอากาศ 2,000W (2 kW)
     ใช้ 5 ชั่วโมง = 10 kWh
```

### ❌ ลืมคูณ CT ratio
```
ผิด: อ่านมิเตอร์ตรง = 100 kWh
ถูก: มี CT 100/5 → 100 × 20 = 2,000 kWh
```

### ❌ คำนวณค่าไฟผิด
```
ผิด: ใช้มิเตอร์เดือนนี้ตรงๆ
ถูก: เดือนนี้ - เดือนที่แล้ว = หน่วยที่ใช้
```

### ❌ เลือกขนาดผิด
```
ผิด: โหลด 40A ใช้มิเตอร์ 15(45)A
ถูก: ใช้มิเตอร์ 30(100)A หรือ 50(100)A
     (Base Current ≥ โหลด)
```

## Smart Meter Features

### ความสามารถ
- วัดได้ทั้ง 2 ทาง (Import/Export)
- บันทึก MD อัตโนมัติ
- แสดง [[Power-Factor-PF]]
- ส่งข้อมูลผ่าน Internet
- ตรวจจับการขโมยไฟ

### ประโยชน์
- อ่านมิเตอร์ทางไกล
- วิเคราะห์การใช้ไฟ
- เตือนเมื่อใช้ไฟมาก
- รองรับ Solar Cell (Net Metering)

## การคำนวณต้นทุนไฟฟ้า

### ตัวอย่างโรงงาน
```
โหลดเฉลี่ย: 100 kW
ทำงาน: 24 ชม./วัน, 30 วัน/เดือน
PF: 0.85

Energy = 100 × 24 × 30 = 72,000 kWh
MD = 100 kW

ค่าหน่วย: 72,000 × 4 = 288,000 บาท
ค่า MD: 100 × 200 = 20,000 บาท
ค่า PF ต่ำ: (0.85 < 0.9) → ปรับ 5,000 บาท
รวม: 313,000 บาท/เดือน

แก้ไข: ติด [[Capacitor-Bank]] → ประหยัด 5,000 บาท
```
