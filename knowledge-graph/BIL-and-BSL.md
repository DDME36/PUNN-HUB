# BIL and BSL

## BIL (Basic Impulse Level)

### คำนิยาม
**BIL** คือระดับแรงดันกระแสฟ้าผ่าสูงสุดที่อุปกรณ์ทนได้ ใช้ทดสอบความแข็งแรงของฉนวน

### หน่วย
- **kV** (peak value)

### ค่ามาตรฐาน
| ระบบ | BIL |
|---|---|
| 22 kV | 125 kV |
| 115 kV | 550 kV |
| 230 kV | 1,050 kV |
| 500 kV | 1,800 kV |

### รูปคลื่น
- **1.2/50 μs**: ขึ้นถึงจุดสูงสุด 1.2 μs, ลดลงครึ่งหนึ่ง 50 μs

## BSL (Basic Switching Level)

### คำนิยาม
**BSL** คือระดับแรงดันกระแสสวิตช์สูงสุดที่อุปกรณ์ทนได้

### รูปคลื่น
- **250/2500 μs**: ช้ากว่า BIL

## การทดสอบ
```
ทดสอบหม้อแปลง 115 kV
BIL = 550 kV
ยิงคลื่น Impulse 1.2/50 μs
ต้องทนได้โดยไม่เสียหาย
```

## การใช้งาน
- ออกแบบฉนวน
- เลือก Surge Arrester
- ป้องกันฟ้าผ่า
