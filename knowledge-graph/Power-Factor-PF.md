# Power Factor (PF) - ตัวประกอบกำลัง

## คำนิยาม
**ตัวประกอบกำลัง (Power Factor)** คืออัตราส่วนระหว่างกำลังจริงที่ใช้งานได้กับกำลังปรากฏทั้งหมด บอกประสิทธิภาพการใช้ไฟฟ้า

## สัญลักษณ์
- **PF** หรือ **cos θ**
- ค่า: **0 ถึง 1** (หรือ 0% ถึง 100%)

## สูตรพื้นฐาน
$$PF = \cos\theta = \frac{P}{S}$$

$$PF = \frac{R}{|Z|}$$

$$PF = \frac{\text{Real Power}}{\text{Apparent Power}}$$

## ความสัมพันธ์กับแนวคิดอื่น

### 🔺 สามเหลี่ยมกำลัง
- [[Real-Power-P]] = [[Apparent-Power-S]] × PF
- [[Reactive-Power-Q]] = [[Apparent-Power-S]] × sin θ
- $S^2 = P^2 + Q^2$

### ⚡ ความสัมพันธ์
- [[Power-Factor-PF]] ↔ [[Impedance-Z]]: $PF = \frac{R}{|Z|}$
- [[Power-Factor-PF]] ↔ มุมเฟส θ
- [[Power-Factor-PF]] ต่ำ → [[Reactive-Power-Q]] สูง

### 🔧 การแก้ไข
- [[Power-Factor-Correction]] → ใช้ [[Capacitor-Bank]]
- ลด [[Reactive-Power-Q]] ล้าหลัง
- เพิ่ม PF ให้ใกล้ 1

## ประเภทของ Power Factor

### 1. Lagging PF (ล้าหลัง)
- กระแสล้าหลังแรงดัน
- โหลดเป็น **Inductive** ([[Inductor-L]])
- ตัวอย่าง: [[Motor]], [[Transformer]]
- $X_L > X_C$

### 2. Leading PF (นำหน้า)
- กระแสนำหน้าแรงดัน
- โหลดเป็น **Capacitive** ([[Capacitor-C]])
- ตัวอย่าง: [[Capacitor-Bank]]
- $X_C > X_L$

### 3. Unity PF (เฟส)
- PF = 1 (100%)
- กระแสและแรงดันเฟสเดียวกัน
- โหลดเป็น **Resistive** ([[Resistor]])
- $X_L = X_C$ (Resonance)

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หา PF จากกำลัง
```
P = 8 kW, S = 10 kVA
PF = P/S = 8/10 = 0.8 (80%)
```

### ตัวอย่าง 2: หา PF จากอิมพีแดนซ์
```
Z = 3 + j4 Ω
R = 3Ω, |Z| = 5Ω
PF = R/|Z| = 3/5 = 0.6 (60%)
```

### ตัวอย่าง 3: หากำลังจริง
```
S = 20 kVA, PF = 0.85
P = S × PF = 20 × 0.85 = 17 kW
```

## ผลกระทบของ PF ต่ำ

### ❌ ปัญหา
1. **กระแสสูง**: $I = \frac{P}{V \times PF}$ → PF ต่ำ → I สูง
2. **สายร้อน**: $P_{loss} = I^2R$ → I สูง → สูญเสียมาก
3. **ค่าปรับ**: การไฟฟ้าปรับเงินถ้า PF < 0.85
4. **ขนาดอุปกรณ์**: ต้องใช้สาย/เบรกเกอร์ใหญ่ขึ้น

### ✅ ประโยชน์ของ PF สูง
1. ลดกระแส → ลดสูญเสีย
2. ลดค่าไฟ
3. ใช้อุปกรณ์ขนาดเล็กลง
4. ระบบเสถียรขึ้น

## เกณฑ์มาตรฐาน
- **PF ≥ 0.85**: ไม่มีค่าปรับ
- **PF < 0.85**: การไฟฟ้าปรับเงิน
- **เป้าหมาย**: PF ≥ 0.95

## การแก้ PF ต่ำ
```
มอเตอร์ 100 kW, PF = 0.7 → ต้องการ PF = 0.95

ขั้นตอน:
1. คำนวณ Q เดิม: Q₁ = P × tan(cos⁻¹(0.7))
2. คำนวณ Q ใหม่: Q₂ = P × tan(cos⁻¹(0.95))
3. ขนาด Capacitor: Q_C = Q₁ - Q₂
4. ติดตั้ง [[Capacitor-Bank]]
```

## ข้อผิดพลาดที่พบบ่อย
❌ **ผิด:** PF = P/V หรือ P/I  
✅ **ถูก:** PF = P/S (กำลังจริง/กำลังปรากฏ)

❌ **ผิด:** PF > 1 ได้  
✅ **ถูก:** PF ≤ 1 เสมอ (0-100%)

❌ **ผิด:** แก้ PF ด้วยการเปลี่ยนมอเตอร์  
✅ **ถูก:** ใช้ [[Capacitor-Bank]] แก้ PF

❌ **ผิด:** PF ต่ำ = มอเตอร์พัง  
✅ **ถูก:** PF ต่ำ = มอเตอร์ต้องการ Q สูง (ปกติของมอเตอร์)

## ตัวอย่างค่า PF ทั่วไป
| อุปกรณ์ | PF โดยประมาณ |
|---|---|
| หลอดไส้ | 1.0 |
| หลอดฟลูออเรสเซนต์ | 0.5-0.6 |
| มอเตอร์ (โหลดเต็ม) | 0.85-0.90 |
| มอเตอร์ (โหลดต่ำ) | 0.3-0.5 |
| หม้อเชื่อม | 0.5-0.7 |
| เตาเหนี่ยวนำ | 0.6-0.8 |
