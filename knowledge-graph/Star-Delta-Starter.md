# Star-Delta Starter - สตาร์ทแบบสตาร์-เดลต้า

## คำนิยาม
**Star-Delta Starter** คือวิธีสตาร์ท [[Motor]] โดยเริ่มต้นด้วย [[Star-Connection]] แล้วเปลี่ยนเป็น [[Delta-Connection]] เมื่อความเร็วใกล้เต็มที่

## หลักการทำงาน

### ขั้นตอนที่ 1: สตาร์ทด้วย Star (Y)
```
ต่อขดลวดแบบ Star
V_coil = V_line / √3
I_line = I_coil
I_start = I_rated / 3
T_start = T_rated / 3
```

### ขั้นตอนที่ 2: เปลี่ยนเป็น Delta (Δ)
```
เมื่อความเร็วถึง 80-90%
เปลี่ยนเป็น Delta
V_coil = V_line
I_line = √3 × I_coil
I = I_rated
T = T_rated
```

## ข้อดี

### 1. ลดกระแสสตาร์ท
```
DOL: I_start = 5-7 × I_rated
Star-Delta: I_start = (5-7)/3 × I_rated ≈ 1.7-2.3 × I_rated

ลดลงเหลือ 1/3 !
```

### 2. ลดแรงดันตก
- กระแสต่ำ → แรงดันตกน้อย
- ไม่รบกวนอุปกรณ์อื่น

### 3. ราคาถูก
- ถูกกว่า [[Soft-Starter]]
- ถูกกว่า [[VSD]]
- ใช้แค่ Contactor 3 ตัว

## ข้อเสีย

### 1. แรงบิดสตาร์ทต่ำ
```
T_start = T_rated / 3

ลดลงเหลือ 1/3 เช่นกัน!
```

### 2. กระแสกระชาก (Transient)
- ตอนเปลี่ยน Star → Delta
- เกิดกระแสกระชากสูง
- อาจสูงกว่า DOL ชั่วขณะ

### 3. ต้องมี 6 สาย
- ต้องดึงสายออกมา 6 เส้น
- U1, U2, V1, V2, W1, W2
- ถ้ามอเตอร์ต่อภายในไม่ได้

## วงจรควบคุม

### อุปกรณ์
- **K1 (Main Contactor)**: ต่อสายเข้ามอเตอร์
- **K2 (Star Contactor)**: ต่อแบบ Star
- **K3 (Delta Contactor)**: ต่อแบบ Delta
- **Timer**: ตั้งเวลาเปลี่ยน

### ลำดับการทำงาน
```
1. กด Start
2. K1 + K2 ทำงาน (Star)
3. รอ 5-10 วินาที (Timer)
4. K2 ดับ
5. K3 ทำงาน (Delta)
6. ทำงานปกติ
```

### Interlock
- K2 และ K3 ต้องไม่ทำงานพร้อมกัน
- ถ้าทำงานพร้อมกัน → ลัดวงจร!

## การตั้งเวลา

### Timer Setting
```
เวลาสตาร์ท = 5-10 วินาที

เร็วเกินไป → มอเตอร์ยังหมุนไม่เร็วพอ
ช้าเกินไป → สิ้นเปลืองพลังงาน
```

### การปรับ
- โหลดเบา: 5-7 วินาที
- โหลดหนัก: 8-10 วินาที
- ทดลองปรับจนเหมาะสม

## เงื่อนไขการใช้

### เหมาะกับ
- มอเตอร์ขนาด 5-200 HP
- โหลดเบา-ปานกลาง
- พัดลม, ปั๊ม, คอมเพรสเซอร์

### ไม่เหมาะกับ
- โหลดหนักตั้งแต่เริ่มต้น
- เครน, รถยก
- สายพานลำเลียงที่บรรทุกหนัก

## การคำนวณ

### กระแสสตาร์ท
$$I_{start(Y-Δ)} = \frac{I_{start(DOL)}}{3}$$

### แรงบิดสตาร์ท
$$T_{start(Y-Δ)} = \frac{T_{start(DOL)}}{3}$$

### ตัวอย่าง
```
มอเตอร์ 30 kW, 380V
I_rated = 55A
I_start(DOL) = 6 × 55 = 330A

Star-Delta:
I_start = 330 / 3 = 110A (ลด 67%)
```

## เปรียบเทียบวิธีสตาร์ท

| วิธี | I_start | T_start | ราคา |
|---|---|---|---|
| DOL | 5-7× | 1.5-2× | ถูกสุด |
| Star-Delta | 1.7-2.3× | 0.5-0.7× | ถูก |
| [[Soft-Starter]] | 2-4× | 0.3-0.8× | ปานกลาง |
| [[VSD]] | 1-1.5× | 0-1.5× | แพง |
