# Eddy Current Loss - การสูญเสียกระแสวน

## คำนิยาม
**การสูญเสียกระแสวน (Eddy Current Loss)** คือการสูญเสียพลังงานในแกนเหล็กเนื่องจากกระแสไฟฟ้าที่เหนี่ยวนำในตัวแกนเองเมื่อฟลักซ์แม่เหล็กเปลี่ยนแปลง

## สาเหตุ

### กฎของฟาราเดย์
```
เมื่อฟลักซ์แม่เหล็กตัดผ่านตัวนำ
→ เกิด EMF เหนี่ยวนำ
→ เกิดกระแสไหลเป็นวงวน (Eddy Current)
→ กระแสนี้ทำให้เกิดความร้อน (I²R)
```

### ลักษณะกระแสวน
- ไหลเป็นวงกลมในแกนเหล็ก
- ตั้งฉากกับทิศทางฟลักซ์
- ยิ่งแกนหนา กระแสยิ่งมาก

## สูตร
$$P_e = K_e f^2 B_{max}^2 t^2 V$$

- Pe = การสูญเสียกระแสวน (W)
- Ke = ค่าคงที่ของวัสดุ
- f = ความถี่ (Hz)
- Bmax = ความหนาแน่นฟลักซ์สูงสุด (T)
- t = ความหนาของแผ่นเหล็ก (m)
- V = ปริมาตรแกน (m³)

## ความสัมพันธ์

### ขึ้นกับความถี่
$$P_e \propto f^2$$
- ความถี่สูง → สูญเสียมากแบบยกกำลังสอง
- ไฟ 60 Hz สูญเสียมากกว่า 50 Hz ถึง 44%

### ขึ้นกับความหนา
$$P_e \propto t^2$$
- แกนหนา 2 เท่า → สูญเสีย 4 เท่า!

## การลดการสูญเสีย

### 1. Lamination (ใช้แผ่นบาง)
```
แทนที่จะใช้เหล็กแท่งตัน
→ ใช้แผ่นเหล็กบางๆ (0.35-0.5 mm)
→ เคลือบฉนวนระหว่างแผ่น
→ กระแสวนลดลงมาก
```

#### ประสิทธิภาพ
- แผ่นบาง 0.35 mm → ลด Pe ได้ 90%
- ยิ่งบางยิ่งดี แต่ราคาแพงขึ้น

### 2. เพิ่มความต้านทาน
- ผสม Silicon (Si) 3-4%
- เพิ่มความต้านทานของเหล็ก
- กระแสวนลดลง

### 3. ลดความถี่
- ใช้ความถี่ต่ำ (50 Hz ดีกว่า 60 Hz)
- แต่ไม่สามารถเปลี่ยนได้ (ขึ้นกับระบบ)

## ในเครื่องจักรไฟฟ้า

### Transformer
- สูญเสียตลอดเวลา
- ประมาณ 70-80% ของ Core Loss
- ใช้แผ่นเหล็กบาง 0.35 mm

### Motor
- เกิดทั้งใน Stator และ Rotor
- Rotor: ความถี่ต่ำกว่า (= Slip × f)
- ใช้แผ่นเหล็กบาง 0.5 mm

### High Frequency
- ที่ความถี่สูง (kHz-MHz)
- ใช้ Ferrite Core แทนเหล็ก
- หรือใช้ Air Core

## ผลกระทบ

### ความร้อน
- แกนร้อนขึ้น
- ต้องระบายความร้อน
- อาจทำลายฉนวน

### ประสิทธิภาพ
- สูญเสียพลังงาน
- ประสิทธิภาพลดลง
