# Slip (s) - สลิป

## คำนิยาม
**สลิป (Slip)** คือความแตกต่างระหว่างความเร็วสนามแม่เหล็กกับความเร็วโรเตอร์ของ [[Motor]] แสดงเป็นเปอร์เซ็นต์

## สัญลักษณ์
- **s** = Slip
- หน่วย: ไม่มีหน่วย หรือ **%**

## สูตร
$$s = \frac{N_s - N_r}{N_s}$$

$$N_r = N_s(1 - s)$$

- Ns = [[Synchronous-Speed]] (rpm)
- Nr = ความเร็วโรเตอร์จริง (rpm)

## ความสัมพันธ์กับแนวคิดอื่น

### ⚙️ มอเตอร์
- [[Slip]] → บอกสภาวะการทำงานของ [[Motor]]
- s < 1 → สภาวะมอเตอร์ (กินไฟ)
- s = 0 → Synchronous Motor
- s > 1 → สภาวะเครื่องกำเนิด (ผลิตไฟ)

### ⚡ ความถี่โรเตอร์
$$f_r = s \times f$$

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หาสลิป
```
N_s = 1,500 rpm, N_r = 1,450 rpm
s = (N_s - N_r) / N_s
s = (1,500 - 1,450) / 1,500
s = 50 / 1,500 = 0.0333 = 3.33%
```

### ตัวอย่าง 2: หา Nr
```
N_s = 1,500 rpm, s = 4% = 0.04
N_r = N_s(1 - s)
N_r = 1,500 × (1 - 0.04)
N_r = 1,500 × 0.96 = 1,440 rpm
```

### ตัวอย่าง 3: ความถี่โรเตอร์
```
f = 50 Hz, s = 0.04
f_r = s × f = 0.04 × 50 = 2 Hz
```

## ค่าสลิปทั่วไป

| สภาวะ | Slip (%) |
|---|---|
| ไม่มีโหลด | 0.5-1% |
| โหลดเต็ม | 3-5% |
| สตาร์ท | 100% |
| Synchronous Motor | 0% |

## สภาวะการทำงาน
- **Nr < Ns** (s > 0): มอเตอร์ - กินไฟไปหมุน
- **Nr = Ns** (s = 0): Synchronous Motor
- **Nr > Ns** (s < 0): เครื่องกำเนิด - หมุนผลิตไฟ
