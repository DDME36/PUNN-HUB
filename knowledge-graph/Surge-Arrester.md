# Surge Arrester - กับดักเสิร์จ

## คำนิยาม
**กับดักเสิร์จ (Surge Arrester)** คืออุปกรณ์ป้องกันระบบไฟฟ้าจากแรงดันกระชากสูง (Surge) ที่เกิดจากฟ้าผ่าหรือการสวิตช์

## หลักการทำงาน

### ปกติ
- ความต้านทานสูงมาก (เกือบ ∞)
- ไม่มีกระแสไหลผ่าน
- ไม่รบกวนระบบ

### เมื่อเกิด Surge
- ความต้านทานลดลงทันที
- ระบายกระแส Surge ลงดิน
- จำกัดแรงดันไม่ให้สูงเกินไป

### หลัง Surge
- กลับเป็นฉนวนอีกครั้ง
- ตัดกระแสตามมา (Follow current)

## ประเภท

### 1. Silicon Carbide (SiC) Type
- รุ่นเก่า
- มีช่องว่าง (Gap)
- ต้องตัดกระแสตามมา

### 2. Metal Oxide (ZnO) Type
- รุ่นใหม่ (นิยมใช้)
- ไม่มีช่องว่าง (Gapless)
- ตัดกระแสตามมาได้เอง

## คุณสมบัติสำคัญ

### 1. MCOV (Maximum Continuous Operating Voltage)
- แรงดันทำงานสูงสุดที่ต่อได้ตลอดเวลา
- เช่น ระบบ 22 kV → MCOV ≈ 18 kV

### 2. Discharge Voltage
- แรงดันที่ปรากฏขณะระบาย Surge
- ต้องต่ำกว่า [[BIL-and-BSL]] ของอุปกรณ์

### 3. Discharge Current
- กระแส Surge ที่ระบายได้
- มาตรฐาน: 5 kA, 10 kA, 20 kA

### 4. Energy Absorption
- พลังงานที่ดูดซับได้
- หน่วย: kJ/kV

## การติดตั้ง

### ตำแหน่ง
- ติดตั้งใกล้อุปกรณ์ที่ต้องการป้องกัน
- ระยะห่างไม่เกิน 10-15 เมตร
- ยิ่งใกล้ยิ่งดี

### การต่อ
```
Line → Surge Arrester → Ground
         |
    อุปกรณ์ที่ป้องกัน
```

### สายต่อลงดิน
- ใช้สายขนาดใหญ่ (≥ 25 mm²)
- ยาวที่สุดไม่เกิน 5 เมตร
- ต่อตรงไปหลักดิน

## การเลือกใช้

### 1. เลือกแรงดันพิกัด
```
V_rated ≥ 1.25 × V_system(L-N)

ตัวอย่าง:
ระบบ 22 kV (L-L)
V_L-N = 22/√3 = 12.7 kV
V_rated ≥ 1.25 × 12.7 = 15.9 kV
→ เลือก 18 kV
```

### 2. เลือก Discharge Current
- ทั่วไป: 5-10 kA
- สายส่งแรงสูง: 10-20 kA
- สถานีไฟฟ้า: 20 kA ขึ้นไป

### 3. ตรวจสอบ BIL
```
Discharge Voltage < BIL ของอุปกรณ์
```

## การทดสอบ

### 1. Insulation Resistance
- วัดด้วย Megger
- ต้องสูง (> 1000 MΩ)

### 2. Leakage Current
- วัดกระแสรั่ว
- ต้องต่ำ (< 1 mA)

### 3. Discharge Test
- ทดสอบการระบาย Surge
- ตรวจสอบ Discharge Voltage

## การบำรุงรักษา

### ตรวจสอบ
- ความสะอาด
- รอยร้าว
- การต่อสายดิน
- Leakage Current

### เปลี่ยน
- ถ้า Leakage Current สูง
- ถ้าเคยระบาย Surge ใหญ่
- ตามอายุการใช้งาน (15-20 ปี)

## ความแตกต่างจาก Lightning Rod

| | Surge Arrester | Lightning Rod |
|---|---|---|
| หน้าที่ | จำกัดแรงดัน | ดึงฟ้าผ่า |
| ตำแหน่ง | ต่อกับสายไฟ | ติดบนหลังคา |
| ทำงาน | ระบายกระแส | นำกระแสลงดิน |
