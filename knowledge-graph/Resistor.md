# Resistor - ตัวต้านทาน

## คำนิยาม
**ตัวต้านทาน (Resistor)** คืออุปกรณ์อิเล็กทรอนิกส์ที่ใช้จำกัดกระแสไฟฟ้าและแบ่งแรงดัน มีค่า [[Resistance-R]] คงที่

## สัญลักษณ์วงจร
```
---[R]--- หรือ ---/\/\/\---
```

## คุณสมบัติ
- ค่าความต้านทาน: Ω (โอห์ม)
- กำลังไฟฟ้า: W (วัตต์) - 1/8W, 1/4W, 1/2W, 1W, 2W, 5W
- ความคลาดเคลื่อน: ±1%, ±5%, ±10%

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ แนวคิดพื้นฐาน
- [[Resistor]] → มีค่า [[Resistance-R]]
- [[Resistor]] → ใช้ [[Ohms-Law]]
- [[Resistor]] → สร้าง [[Real-Power-P]] (ความร้อน)

### 🔌 การต่อวงจร
- [[Series-Connection]]: $R_{total} = R_1 + R_2 + R_3$
- [[Parallel-Connection]]: $\frac{1}{R_{total}} = \frac{1}{R_1} + \frac{1}{R_2}$

### 🌡️ ผลกระทบ
- แปลงพลังงานไฟฟ้า → ความร้อน
- $P = I^2R$ (ต้องเลือกกำลังให้เหมาะสม)

### ตัวอย่าง
```
น้ำตาล-ดำ-แดง-ทอง = 10×100Ω ±5% = 1kΩ ±5%
เหลือง-ม่วง-ส้ม-น้ำตาล = 47×1kΩ ±1% = 47kΩ ±1%
```

## ประเภทของตัวต้านทาน
1. **Carbon Film**: ราคาถูก, ใช้ทั่วไป
2. **Metal Film**: แม่นยำกว่า, เสียงรบกวนต่ำ
3. **Wire Wound**: กำลังสูง, ความแม่นยำสูง
4. **SMD**: ขนาดเล็ก, ใช้ใน PCB

## การเลือกใช้
1. **ค่าความต้านทาน**: ตามการออกแบบวงจร
2. **กำลังไฟฟ้า**: $P = I^2R$ (เลือกสูงกว่าที่คำนวณได้ 2 เท่า)
3. **ความคลาดเคลื่อน**: ±1% สำหรับวงจรแม่นยำ

## ข้อผิดพลาดที่พบบ่อย
❌ **ผิด:** ใช้ตัวต้านทาน 1/4W กับวงจรที่ใช้ 0.3W  
✅ **ถูก:** ต้องเลือกกำลังสูงกว่า → ใช้ 1/2W หรือ 1W

❌ **ผิด:** ต่ออนุกรม 2 ตัว 10Ω ได้ 20Ω แต่กำลังเท่าเดิม  
✅ **ถูก:** กำลังรวมเพิ่มเป็น 2 เท่า
