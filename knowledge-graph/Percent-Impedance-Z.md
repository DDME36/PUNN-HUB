# Percent Impedance (%Z)

## คำนิยาม
**%Z (Percent Impedance)** คืออิมพีแดนซ์ของ [[Transformer]] แสดงเป็นเปอร์เซ็นต์ของแรงดันตกคร่อมเมื่อไหลกระแสเต็มพิกัด

## ความหมาย
%Z = 5% หมายความว่า:
- เมื่อไหลกระแสเต็มพิกัด
- แรงดันตกใน [[Transformer]] = 5% ของแรงดันเนมเพลต

## การใช้งาน

### หากระแสลัดวงจร
$$I_{sc} = \frac{I_{rated}}{\%Z} \times 100$$

ตัวอย่าง:
```
หม้อแปลง 100 kVA, 380V, %Z = 5%
I_rated = 100,000/(√3 × 380) = 152A

I_sc = (152 / 5) × 100 = 3,040A
```

### ใน Per-Unit System
$$Z_{pu} = \frac{\%Z}{100}$$

%Z = 5% → Zpu = 0.05 pu

## ค่าทั่วไป
- หม้อแปลงเล็ก: 3-5%
- หม้อแปลงกลาง: 5-8%
- หม้อแปลงใหญ่: 8-15%
