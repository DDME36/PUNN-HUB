# Inductive Reactance (XL) - รีแอคแตนซ์เหนี่ยวนำ

## คำนิยาม
**รีแอคแตนซ์เหนี่ยวนำ (Inductive Reactance)** คือความต้านทานที่เกิดจาก [[Inductor-L]] ในวงจร AC ซึ่งเพิ่มขึ้นตามความถี่

## สัญลักษณ์
- **XL** = Inductive Reactance
- หน่วย: **Ohm (Ω)**

## สูตร
$$X_L = 2\pi f L = \omega L$$

- f = [[Frequency-f]] (Hz)
- L = ความเหนี่ยวนำ (H)
- ω = ความถี่เชิงมุม (rad/s)

## ความสัมพันธ์กับแนวคิดอื่น

### ⚡ พื้นฐาน
- [[Inductor-L]] → สร้าง [[Inductive-Reactance-XL]]
- [[Inductive-Reactance-XL]] → ส่วนหนึ่งของ [[Reactance]]
- [[Inductive-Reactance-XL]] → ส่วน $+jX_L$ ใน [[Impedance-Z]]

### 🔺 กำลังและเฟส
- [[Inductive-Reactance-XL]] → สร้าง [[Reactive-Power-Q]] ล้าหลัง
- กระแสล้าหลังแรงดัน 90°
- [[Power-Factor-PF]] ล้าหลัง (Lagging)

### ⚙️ การใช้งาน
- [[Motor]] → มี XL สูง
- [[Transformer]] → มี XL
- Choke, Filter

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หา XL
```
L = 0.1 H, f = 50 Hz
X_L = 2πfL = 2 × 3.14159 × 50 × 0.1
X_L = 31.4 Ω
```

### ตัวอย่าง 2: ผลของความถี่
```
L = 0.1 H
f = 50 Hz → X_L = 31.4 Ω
f = 60 Hz → X_L = 37.7 Ω
f = 100 Hz → X_L = 62.8 Ω
```

### ตัวอย่าง 3: หา L
```
X_L = 100 Ω, f = 50 Hz
L = X_L / (2πf) = 100 / (2π × 50)
L = 0.318 H = 318 mH
```

## ความสัมพันธ์กับความถี่

| ความถี่ | XL | พฤติกรรม |
|---|---|---|
| DC (0 Hz) | 0 Ω | Short circuit |
| ต่ำ (50 Hz) | ต่ำ | ต้านน้อย |
| สูง (1 MHz) | สูงมาก | ต้านมาก |

**สรุป:** ยิ่งความถี่สูง → XL ยิ่งสูง

## เฟสเซอร์
```
V นำหน้า I 90°
หรือ I ล้าหลัง V 90°

V = V∠0°
I = I∠-90°
```

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** XL คงที่  
✅ **ถูก:** XL ขึ้นกับความถี่ (XL ∝ f)

❌ **ผิด:** ใน DC, XL = ∞  
✅ **ถูก:** ใน DC, XL = 0 (ทำตัวเหมือนสายไฟ)

❌ **ผิด:** XL ทำให้กระแสนำหน้า  
✅ **ถูก:** XL ทำให้กระแสล้าหลัง 90°
