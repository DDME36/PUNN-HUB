# Node Voltage Analysis - การวิเคราะห์แรงดันจุดต่อ

## คำนิยาม
**Node Voltage Analysis** หรือ **Nodal Analysis** คือวิธีวิเคราะห์วงจรโดยใช้ [[Kirchhoffs-Laws|KCL]] ที่จุดต่อ (Node) เพื่อหาแรงดันที่จุดต่อทั้งหมด

## หลักการ
- เลือก **Reference Node** (Ground, V = 0)
- กำหนดแรงดันที่จุดต่ออื่นๆ (V₁, V₂, V₃, ...)
- เขียนสมการ KCL ที่แต่ละจุดต่อ
- แก้สมการหาแรงดัน

## ขั้นตอน

### 1. เลือก Reference Node
```
เลือกจุดที่:
- มีสาดเชื่อมมากที่สุด
- ต่อลงดิน (Ground)
- สะดวกในการคำนวณ
```

### 2. กำหนดแรงดันจุดต่อ
```
Reference Node: V = 0
จุดต่ออื่นๆ: V₁, V₂, V₃, ...
```

### 3. เขียนสมการ KCL
```
ที่จุดต่อ A:
∑ I_in = ∑ I_out

หรือ

∑ I = 0
```

### 4. แปลงกระแสเป็นแรงดัน
```
I = (V_A - V_B) / R

ใช้ [[Ohms-Law]]
```

## ตัวอย่างการใช้งาน

### ตัวอย่าง 1: วงจร 2 จุดต่อ
```
วงจร:
V_s = 12V
R₁ = 2Ω (V_s → V₁)
R₂ = 3Ω (V₁ → Ground)
R₃ = 4Ω (V₁ → Ground)

Reference: Ground (V = 0)
Unknown: V₁

KCL ที่ V₁:
(V_s - V₁)/R₁ = V₁/R₂ + V₁/R₃
(12 - V₁)/2 = V₁/3 + V₁/4

แก้สมการ:
6 - 0.5V₁ = 0.333V₁ + 0.25V₁
6 = 1.083V₁
V₁ = 5.54V
```

### ตัวอย่าง 2: วงจร 3 จุดต่อ
```
วงจร:
I_s = 2A (แหล่งจ่ายกระแส)
R₁ = 4Ω (V₁ → V₂)
R₂ = 2Ω (V₁ → Ground)
R₃ = 6Ω (V₂ → Ground)

Reference: Ground
Unknown: V₁, V₂

KCL ที่ V₁:
I_s = V₁/R₂ + (V₁ - V₂)/R₁
2 = V₁/2 + (V₁ - V₂)/4

KCL ที่ V₂:
(V₁ - V₂)/R₁ = V₂/R₃
(V₁ - V₂)/4 = V₂/6

แก้สมการ 2 ตัวแปร:
V₁ = 6V
V₂ = 3.6V
```

## เทคนิคพิเศษ

### Super Node
เมื่อมีแหล่งจ่ายแรงดันระหว่าง 2 จุดต่อ

```
V₁ - V₂ = V_s (ทราบแล้ว)

เขียน KCL รอบ Super Node:
∑ I_in = ∑ I_out (ของทั้ง 2 จุด)
```

### แหล่งจ่ายกระแส
```
ง่ายกว่าแหล่งจ่ายแรงดัน!
เขียน I_s ลงในสมการ KCL ตรงๆ
```

## ข้อดี-ข้อเสีย

### ข้อดี
- เหมาะกับวงจรที่มีแหล่งจ่ายกระแส
- สมการน้อยกว่า Mesh (n-1 สมการ)
- ใช้กับวงจรที่ไม่เป็น Planar ได้

### ข้อเสีย
- ยากกับแหล่งจ่ายแรงดัน (ต้องใช้ Super Node)
- ต้องระวังเครื่องหมาย

## การใช้กับ AC Circuit

### วงจร AC
```
ใช้ [[Complex-Numbers-AC]]
Ṽ = V∠θ
Z̃ = R + jX

KCL: ∑ Ĩ = 0
Ĩ = (Ṽ_A - Ṽ_B) / Z̃
```

### ตัวอย่าง AC
```
Ṽ_s = 220∠0° V
Z₁ = 3 + j4 Ω
Z₂ = 5 - j2 Ω

KCL ที่ V₁:
(Ṽ_s - Ṽ₁)/Z₁ = Ṽ₁/Z₂

แก้สมการ Complex
```

## ข้อผิดพลาดที่พบบ่อย

### ❌ เครื่องหมายผิด
```
ผิด: I = (V_B - V_A) / R (ทิศทางผิด)
ถูก: I = (V_A - V_B) / R (จาก A → B)
```

### ❌ ลืม Reference Node
```
ผิด: กำหนดแรงดันทุกจุด
ถูก: เลือก 1 จุดเป็น Reference (V = 0)
```

### ❌ นับสมการผิด
```
n = จำนวนจุดต่อทั้งหมด
สมการที่ต้องการ = n - 1
(ไม่นับ Reference Node)
```

### ❌ แหล่งจ่ายแรงดัน
```
ผิด: เขียน KCL ที่จุดต่อที่มี V_s
ถูก: ใช้ Super Node

หรือ

แปลง V_s → I_s (ถ้าทำได้)
```

## การประยุกต์ใช้

### 1. วงจรซับซ้อน
- วิเคราะห์วงจรที่มีหลายจุดต่อ
- หาแรงดันที่จุดต่อต่างๆ

### 2. Power System
- ใช้ใน [[Load-Flow-Analysis]]
- หาแรงดันที่ Bus ต่างๆ

### 3. Electronic Circuit
- วิเคราะห์วงจร Op-Amp
- วิเคราะห์วงจร Transistor

## ตัวอย่างโจทย์ยาก

### โจทย์: วงจรแบบสมมาตร
```
วงจร Bridge:
V_s = 24V
R₁ = R₂ = R₃ = R₄ = 10Ω
R₅ = 5Ω (ตรงกลาง)

หา: V₁, V₂, V₃

วิธีทำ:
1. Reference: Ground (ล่างซ้าย)
2. V₁ (บนซ้าย), V₂ (บนขวา), V₃ (ล่างขวา)
3. เขียน KCL ที่ V₁, V₂, V₃
4. แก้สมการ 3 ตัวแปร

ผลลัพธ์:
V₁ = 16V
V₂ = 16V
V₃ = 8V
```

## Tips & Tricks

### 1. เลือก Reference ดีๆ
- เลือกจุดที่มีสายเชื่อมมากที่สุด
- ลดจำนวนสมการ

### 2. ใช้ Symmetry
- ถ้าวงจรสมมาตร → แรงดันเท่ากัน
- ลดตัวแปร

### 3. ตรวจสอบคำตอบ
- ใช้ KVL ตรวจสอบ
- ใช้ [[Ohms-Law]] ตรวจสอบกระแส
