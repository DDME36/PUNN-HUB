# Nameplate Reading - การอ่านเนมเพลต

## คำนิยาม
**การอ่านเนมเพลต (Nameplate Reading)** คือการอ่านข้อมูลจากป้ายติดบนอุปกรณ์ไฟฟ้า เช่น [[Motor]], [[Transformer]]

## เนมเพลตมอเตอร์

### ตัวอย่าง
```
10 HP (7.5 kW)
380V, 3-Phase, 50Hz
15A, 1450 rpm
PF = 0.85, Eff = 90%
Class F, IP55
```

### การอ่าน
- **10 HP (7.5 kW)**: กำลังขาออกที่เพลา (Output)
- **380V**: แรงดันสาย
- **15A**: กระแสสาย (โหลดเต็ม)
- **1450 rpm**: ความเร็วโหลดเต็ม
- **PF = 0.85**: ตัวประกอบกำลัง
- **Eff = 90%**: ประสิทธิภาพ
- **Class F**: ฉนวนทน 155°C
- **IP55**: ป้องกันฝุ่นและน้ำ

## เนมเพลตหม้อแปลง

### ตัวอย่าง
```
100 kVA
22kV / 380V
50 Hz
%Z = 4%
```

### การอ่าน
- **100 kVA**: กำลังปรากฏสูงสุด
- **22kV / 380V**: แรงดันปฐมภูมิ / ทุติยภูมิ
- **50 Hz**: ความถี่
- **%Z = 4%**: เปอร์เซ็นต์อิมพีแดนซ์

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** kW บนมอเตอร์ = กำลังไฟฟ้าที่กิน  
✅ **ถูก:** kW = กำลังกลที่เพลา (Output)

❌ **ผิด:** kVA บนหม้อแปลง = kW  
✅ **ถูก:** kVA = Apparent Power, kW = kVA × PF
