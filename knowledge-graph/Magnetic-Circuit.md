# Magnetic Circuit - วงจรแม่เหล็ก

## คำนิยาม
**วงจรแม่เหล็ก (Magnetic Circuit)** คือเส้นทางของฟลักซ์แม่เหล็กในแกนเหล็ก คล้ายกับวงจรไฟฟ้า

## พารามิเตอร์หลัก

### 1. MMF (Magnetomotive Force)
$$\mathcal{F} = N \times I$$
- N = จำนวนรอบขดลวด
- I = กระแส (A)
- หน่วย: **Ampere-turns (A·t)**

### 2. Flux (Φ)
$$\Phi = B \times A$$
- B = ความหนาแน่นฟลักซ์ (T)
- A = พื้นที่หน้าตัด (m²)
- หน่วย: **Weber (Wb)**

### 3. Reluctance (R)
$$\mathcal{R} = \frac{l}{\mu A}$$
- l = ความยาวเส้นทาง (m)
- μ = ความซาบซึมแม่เหล็ก (H/m)
- A = พื้นที่หน้าตัด (m²)
- หน่วย: **A·t/Wb**

## กฎของโอห์มแม่เหล็ก
$$\Phi = \frac{\mathcal{F}}{\mathcal{R}}$$

เทียบกับวงจรไฟฟ้า: $I = \frac{V}{R}$

## ความเหนี่ยวนำ
$$L = \frac{N\Phi}{I}$$

## ตัวอย่างการคำนวณ

### ตัวอย่าง 1: หา MMF
```
N = 500 รอบ, I = 2A
F = N × I = 500 × 2 = 1,000 A·t
```

### ตัวอย่าง 2: หา Reluctance
```
l = 0.5 m, A = 10 cm² = 10 × 10⁻⁴ m²
μ = 1,000 μ₀ = 1,000 × 4π × 10⁻⁷

R = l / (μA)
R = 0.5 / (1.257 × 10⁻³ × 10 × 10⁻⁴)
R = 397,887 A·t/Wb
```

## ข้อผิดพลาดที่พบบ่อย

❌ **ผิด:** ลืมแปลง cm² เป็น m²  
✅ **ถูก:** 1 cm² = 10⁻⁴ m² (คูณ 10⁻⁴)

❌ **ผิด:** R ขึ้นกับจำนวนรอบ  
✅ **ถูก:** R ขึ้นกับแกนเหล็ก ไม่เกี่ยวกับ N
