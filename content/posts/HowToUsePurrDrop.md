# 🛠 วิธีส่งไฟล์ (App-to-Screen)

## 1️⃣ เปิดหน้าเว็บด้วยเครื่องทั้งสองฝั่ง

ไม่ว่าจะเป็น:
- 📱 มือถือ สมาร์ทโฟน
- 💻 คอมพิวเตอร์ หรือ โน๊ตบุ๊ค
- 📺 แท็บเล็ต ไอแพด

![ตัวอย่างการแสดงผลบนอุปกรณ์](/images/posts/HowToUsePurrDrop/img-4.png)

เมื่อเปิดครบทั้งสองเครื่องแล้ว ระบบจะทำการค้นหาอุปกรณ์ของกันและกัน แล้วแสดงออกมาเป็นตัวละครให้เห็นทันทีบนหน้าต่าง!

---

## 2️⃣ เลือกโหมดการเชื่อมต่อ

คุณสามารถเลือกได้ 3 โหมดจากเมนูด้านบน 👇

![โหมดการค้นหา](/images/posts/HowToUsePurrDrop/img-5.png)

### 🌍 **Public Mode (สาธารณะ)**
- เห็นทุกคนที่กำลังเปิดเว็บใช้งาน Public Mode เหมือนกัน
- เหมาะกับการทดสอบใช้งานไวๆ แต่ **ไม่แนะนำ** หากต้องการความเป็นส่วนตัว

### 📶 **WiFi Mode (วงแลนเดียวกัน)**
- เห็นเฉพาะเครื่องที่เชื่อมต่อวง WiFi หรือ LAN เดียวกัน
- เหมาะกับการใช้งานในบ้านหรือออฟฟิศ (ไม่ซับซ้อน ไม่ต้องใช้รหัส)

### 🔐 **Private Mode (แบบส่วนตัวห้องลับ) — แนะนำ!**
- ต้องใช้ **รหัสห้อง 5 หลัก** เพื่อเจอกันเท่านั้น
- ปลอดภัยขั้นสุด สามารถตั้งรหัสผ่านล็อกห้องเพิ่มได้อีกชั้น

---


## 3️⃣ วิธีเริ่มส่งไฟล์

### 📌 วิธีที่ 1 — คลิกเลือกไฟล์
1. แตะ หรือ คลิกที่ **ตัวละครของเครื่องปลายทาง** ที่ปรากฏบนจอ
2. เลือกไฟล์ที่ต้องการส่งจากระบบของคุณ
3. เครื่องปลายทางจะเด้งข้อความเตือนว่า “มีไฟล์ส่งบรรณาการเข้ามา”
4. ให้กด **“รับเลย!”** ไฟล์ก็จะดาวน์โหลดทันที

![ตัวอย่างหน้าต่างรับไฟล์](/images/posts/HowToUsePurrDrop/img-6.jpg)

---

### 📌 วิธีที่ 2 — Drag & Drop (ลากวาง)
1. ลากไฟล์จากโฟลเดอร์ในคอมพิวเตอร์ของคุณ
2. วางลากลงไปเหนี่ยวใส่ **ตัวละครของอีกเครื่องแบบจังๆ**
3. เครื่องปลายทางกดยืนยันรับไฟล์ จบ! ง่ายสุดๆ

![ตัวอย่างการลากไฟล์วาง](/images/posts/HowToUsePurrDrop/img-7.png)

*แถมให้:* หากคุณส่งทีละหลายๆ ไฟล์ในรอบเดียว ระบบจะทำการจัดแพ็กไฟล์ทั้งหมดเป็นไฟล์ **.ZIP** ให้โดยอัตโนมัติ เพื่อความสะดวกในการโหลด!


# 🔒 ความปลอดภัยที่เข้าใจง่าย 100%

## ไม่แวะพักที่เซิร์ฟเวอร์
ไฟล์ของคุณจะถูกส่งวิ่งเป็นทางตรงระหว่างอุปกรณ์ (Peer-to-Peer) ไม่มีการกักเก็บไฟล์ค้างไว้บน Server ใดๆ ทั้งสิ้น

## เข้ารหัสแน่นหนาระหว่างทาง
ขับเคลื่อนด้วยเทคโนโลยี WebRTC + DTLS ข้อมูลของคุณจะถูกเข้ารหัสระดับเดียวกับแอปแชทชั้นนำ ป้องกันการดักจับข้อมูลระหว่างทางได้สบายใจ

## Private Mode หายห่วง
คุณสามารถเปิดโหมด Private ด้วยรหัสห้อง 5 หลัก และหากยังกังวลอยู่ ก็ตั้งรหัสผ่าน Password ล็อกห้องซ้อนทับอีกรอบได้! ไม่มีทางที่ใครจะแอบเข้ามาดึงไฟล์ของคุณไปได้แน่นอน


# ⚡ ความเร็วในการส่ง

- 🏠 **ใช้วง WiFi เดียวกัน:** ส่งไวระดับกระสุนปืน ยิ่งเราเตอร์เร็ว ยิ่งโหลดปรี๊ด 
- 🌐 **คนละ WiFi หรือใช้ 4G/5G:** ส่งได้ปกติ แต่ความเร็วจะขึ้นอยู่กับแพ็กเกจอินเทอร์เน็ตของทั้งคู่
- 📦 **ข้อจำกัดเรื่องขนาดไฟล์:** ไม่มี! คุณสามารถส่งไฟล์ใหญ่ระดับ GB ได้สบายมาก ตราบใดที่เปิดหน้าต่างเว็บทิ้งไว้ไม่ให้เครื่องหลับ

# 📲 รองรับรูปแบบ Progressive Web App (PWA)

## ติดตั้งเป็นแอปในเครื่องได้เลย!
PurrDrop ถูกพัฒนาในรูปแบบ **PWA** เต็มสูบ หมายความว่า:
- 📱 กดปุ่ม Add to Home Screen เพื่อติดตั้งลงหน้าจอมือถือ/แท็บเล็ตได้เลย 
- 🧩 มีไอคอนแอพเหมือนแอปทั่วไปแบบเป๊ะๆ
- 🖥 เวลาเปิดใช้งานจะได้ Layout แบบเต็มหน้าจอ (ไม่มีแถบ Address ของเบราเซอร์กวนตา)

ไม่ต้องฝ่าฟันไปเข้า App Store หรือ Play Store ให้ยุ่งยาก ได้แอปดีๆ มาติดเครื่องได้ทันใจ
